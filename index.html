利未记对现代基督徒的意义

发表于 2017-08-31 | 分类于 圣经解读 | | 阅读次数



许多弟兄姊妹读利未记的时候，觉得读不下去：一方面是因为其中许多的献祭礼仪、程序，洁净的规条令人觉得遥远陌生又枯燥乏味，远不如五经中记载的许多故事精彩；另一方面是认为这些礼仪、规条是旧约的律法，是过时的，对今天活在新约之下的基督徒已经不适用了。当我们这样来看利未记的时候，就是误解了利未记，而且错失了利未记作为神的启示的鲜活意义。

利未记是摩西五经的第三卷，是最中间的那一卷书；从位置来看，这卷书是整个摩西五经的核心。如果把摩西五经当做是旧约信仰的根基的话，那么利未记就是这个根基的核心；利未记的重要性可见一斑。那么，我们如何来解读利未记呢？这些两三千年前的礼仪、规条于今天的你我有什么意义呢？

今日圣经小程序

近日，今日佳音的细心粉丝会在我们的公号上看到一个消息说，公众号关联小程序通知。如果你点进去，就会眼前一亮，看到一个神奇的新东西：今日圣经。

惊艳的那一刻，你会问，这是什么东东？

这是个小程序，可能有很多小伙伴们，还不知道小程序是什么，请看文后小编的介绍和短评1。

作为一个基督徒媒体机构，我们常常很努力的开动小脑筋，想如何使用新媒体把耶稣的宝贵福音更好的传扬出去，好和其他伙伴们一起完成大使命。这是我们最最开心的事！于是，我们一开始做广播，接着做网站；后来微博火了，我们就去做微博；智能手机很多人用，我们开发手机应用；大家都用微信，我们就经营微信公号：今日佳音wxbible；现在，小程序有人用，于是就有了：今日圣经小程序。哒哒哒…

所以啊，今日圣经小程序对很多人来说，貌似横空出世，其实并不是一个偶然。这是我们用心体会上帝爱世人，希望“人人得救，不愿一人沉沦”的心肠，并一直努力回应上帝在这个时代给我们呼召的结果。

工具与新媒体的关系

发表于 2017-08-22 | 分类于 工具控 | | 阅读次数

1. 传统媒体的关系：媒体与工具无关。

媒体和工具是分开的，一般媒体人（报社、杂志社等）并不拥有印刷厂设备和印刷技术。但工具的革新会影响内容的创作，比如彩色印刷就能促进作者（媒体）的创作。
内容和形式的分离，使得媒体人考虑的面比较窄和单纯，比如：纯文字，但可能也因此练就了他们的文字功底。杰出的作者，能用文字营造出绝佳的画面，是其它的媒体无法企及的。

2. 新媒体：媒体掌握工具

媒体除了生产内容，他还要会使用工具，才能把内容传播出去。比如：用编辑器排版，我把公号作者和后台编辑视为一个媒体单位。
反过来，工具造就媒体。现在最大的媒体，可能是facebook，然后是微信。这两者一开始都是工具，但后来成了最大的媒体。现在想做媒体的人，都会想努力提供一个工具，比如：简书，比如app。
两者是相辅相成的，工具成就媒体，媒体也成就工具。没有人用的Facebook或者微信，即使功能再强悍也无济于事。没有人
有人先把工具做好了，现在布局媒体；是从工具到媒体。
有人先把媒体做好了，就要考虑工具实现；是从媒体到工具。

媒体单位要真正强大，她的媒体属性和工具属性缺一不可。

哦，手机，我怎能不忘记你！

发表于 2017-08-17 | 分类于 灵修笔记 | | 阅读次数

作为一个媒体人和工具控患者，我自己也常常反省媒体和工具带来的负面影响。今天读到梅顿《学习一门爱的语言》的灵修短文，说到他去外面灵修看到的场景，他描写苍蝇，描写小马驹，描写死的昆虫，他还提到当他在被按立神父的时候，还匆忙跑去给岛红鸟搭了一个窝。

我突然意识到手机和网络对现代人的危害在哪里，就是让人不能活在此时此刻、此景此情，让人与当下的处境剥离。不乏有这样的场景：一群年轻人参加Party，大家都拿出手机拍照，然后上传照片到网上，接着回复网上的朋友的点赞、评论，不亦乐乎；却对眼前的欢庆场面视若无睹。还有，更心酸的，两个恋人出去约会，不是四目相对，而是各盯着手机，在手机上传递各种表情，乐在其中。
