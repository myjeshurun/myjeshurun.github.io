80后天才程序员自杀：每一场成功的骗局，都有“内奸”在呼应

发表于 2017-09-11 | 分类于 时事评论 | | 阅读次数

天才程序员的悲剧之“死”

苏享茂，福建建瓯人，一个靠着自己的努力和才干而小有成就的程序员，所写Wephone应用拥有3千万的海外用户，这个时代个人奋斗的典型人物，因陷入一场精心策划婚姻的骗局，结果钱财被勒索、事业受威胁，绝望之下，从高楼阳台一跃而下，当场死亡。

看到这样的消息，心里说不出的难过、悲伤；人间最令人痛心的一件事是青翠的生命之树，突然被生生砍断；如同壮阔的交响乐戛然而止，时间从此定格，悲怆之情油然而生。若是天灾横祸等人无法左右的因素，人更多的是哀痛、哀痛，好让时间来治愈；但若是有人的因素，除了哀痛、叹息，自会有许多的反省，期盼能从中学习教训。只是人能挖出更多的事实真相，帮助我们更了解事件的来龙去脉；又有谁能挖出人内在心灵的真相，并给我们力量去避免同样的悲剧再次发生。

每一个成功的骗局，都有“内奸”在呼应

苏享茂的自杀前因后果似乎不难理解，越来越多的证据和分析显示，他落入了一个诈骗团伙的圈套，心灵的弱点被抓住，一步一步被钳制套牢，最终被逼上了绝路。网上除了痛骂翟女的黑心狠毒之外，更多的是对苏享茂作为一个高智商技术天才却在感情婚姻上轻易受骗的叹息。人们一方面痛恨邪恶之人的歹毒，另一方面又勉励无辜良善之人处事要小心，懂得自保。这些都无比必要，但问题是，这就足够了吗？人生在世，多少人没有被欺骗过？我们又有多少人能自信满满的说，靠着自己的谨慎和聪明避开那冲着我们而来的生命骗局呢？

我清楚的记得上高一的时候一次被骗钱的经历。那是20多年前的春节，我和小伙伴们去镇的市场上看热闹。一个江湖郎中坐在地上玩杂耍，不少小朋友围观。我也跟着去看。过了一会，那个人突然停下来，问围观的人说：“如果你们真的给我一块钱，把钱放在放在碗里扣住，我会加上1块钱还给你们吗？”他盯着我们看，有些人就说：“我信。”然后，就拿着一块钱放上去，我觉得自己应该相信别人。相信别人是一种美德，不是吗？再说了，我看这个人挺可信的，一块钱能测试一个人可不可靠也不错，我心里想。于是，一开始，就因为觉得这很好玩，我给了他一块钱。然后，他很认真的问大家：“你们真的愿意把这些钱给我吗？”得到大家的回复，不出所料，他把钱加上一块钱给我们了。我好开心，不仅赚到了钱，而且觉得自己的眼光不错，没有看错人，还勇敢的信了一次别人（我当然根本就不知道信任也是需要有基础的，不然相信就是盲信，迷信）。就在我沉浸在自我感觉良好的情绪中的时候，那个郎中又说话了：“你们看，我说话算数吧。你们这下相信了吧。只要你们愿意真心的把钱给我，我不仅会还给你们，还会奖赏你们。你们想不想再试一次？5块钱，我给2块钱。”我开始觉得这个人很有意思，似乎要用钱来换取别人对他的信任。为什么不成全他呢？而且，白白多赚5块钱呢？后来，他又试了10块钱，给5块。虽然我有些犹豫，但白白多赚5块钱的诱惑盖住了其它的一切。等到最后，他说20块钱给10块的时候，我已经完全信任他了。等我们都放好了钱，他同样问：你们真的愿意把钱给我吗？周围几个放钱的孩子们，包括我在内，都很开心的说：“愿意。”他又重复问了一句，我感觉到大家有些犹豫的，但说出去的话怎么能收回来呢？所有的人都说：“愿意。”他说：“太好了。谢谢你们，孩子们。谢谢你们真心的要给我钱，那这次我就不客气了。”就这样，他把钱都收走了。我一下傻了眼了，愣在那里，才知道一切都是圈套，都是骗局。我无话可说，被骗的心服口服，是我自己说心甘情愿把钱送给别人。20块钱，那是我那一年所有的压岁钱，上高中一个月的零花钱，就这样拱手给了一个骗子；心里很不是滋味，但只好认栽。

这件事我一直埋在心里，从来不好意思跟别人讲，更不敢跟我父母讲让他们帮我去要回那些被骗的钱；他们肯定会训斥我，说我脑袋进水了，读书读到马背上去了。这件事在我儿时的记忆中留下了深刻的印象，我看到自己的不谙世事；自以为是信任别人，其实是盲目无知；看到自己隐秘处的懒惰、贪婪，想着捡从天上掉下来的馅饼；还看到自己隐藏的虚荣，别人看到我愿意给别人钱，我愿意信任别人，我是一个多么好的人啊；以及我的害怕，我的羞耻：我不能让别人知道我被骗了。我心灵深处所有的软弱：无知、盲目、贪心、虚荣、羞耻心，在一个老练的骗子面前，如同穿着新装的皇帝，裸露无疑，我又如何能不上钩呢？

另一场骗局，另一个可怕的结局

这虽是一个很久远的故事，但一直启迪着我。我看到：在一场成功的骗局背后，都是有内奸呼应的，那个内奸就是我们内心的软弱。这个软弱，在基督教的术语里，叫做罪，这罪是指对上帝旨意的偏离甚至背离。而这个罪的由来也是始于一场精心设计的骗局，这是许多人很熟悉的记载在创世记第3章的故事。

上帝创造了亚当夏娃之后，把他们放置在伊甸园里，他们结了婚，成为一体了。夫妻俩在伊甸园辛勤开垦，衣食无忧，也与上帝常常见面，日子过得很滋润。在无比美丽的伊甸园中，上帝只给了他们一个象征着祂自己同在的命令：“园中所有树上的果子都可以吃，唯有那分别善恶树上的果子不可以吃，因为你吃的日子必定死。”有一天，蛇来了，牠是有备而来的。因为：

  1. 牠知道弱点在哪里。牠首先找到夏娃。牠觉得夏娃是比较软弱的一方，一来比较容易靠近，二来比较感性比较容易相信别人。
  2. 牠看起来很良善，很寻求真理。蛇说：“神岂是真说不许你们吃园中所有树上的果子吗？”祂的动机不是要明白真理，而是要试探人，引人离开真理，相信牠的谎言。
  3. 牠看起来很体贴夏娃，很站在她这一边为她的好处考虑。蛇说：“你们不一定死；因为神知道，你们吃的日子眼睛就明亮了，你们便如神能知道善恶。” 牠没有说出来的话是，重要的不是知道善恶，而是行出善恶。人一知道善恶，却没有行善的能力，结果是可怕的。

而夏娃就是这样一步一步上钩的：

  1. 对真理的认识开始模糊。唯有园中那棵树上的果子，神曾说：”你们不可以吃，也不可以摸，免得你们死。”神其实只说了“不可以吃。” 增加上帝的话语，看起来似乎更保险，却是危险的。
  2. 脱离亲密无间的丈夫，让自己暴露在外界陌生人的引诱之中。如果我们把亚当夏娃当做一体，我们可以这么来应用：人脱离了自己的理性，只让感性来做决定；或者人脱离了一般的常识，只让自己的直觉判断做决定；或者人脱离了最熟悉的群体的智慧，只让自己一个人来做决定。
  3. 沉浸在诱惑之中。夏娃见那棵树的果子好做食物，也悦人的眼目，且是可喜爱的，能使人有智慧。骗子是最能挑起人里面的欲望的，满足自己的欲望又何尝不可呢？肉体的欲望，眼目的欲望，今生的骄傲（拥有它就了不起了，就和神一样有智慧了），这个世界是座欲望之城，吞噬着人的灵魂。
  4. 难以抗拒的行动。夏娃就摘下果子吃了。
  5. 拉丈夫下水。又给她丈夫，她丈夫也吃了。泥足深陷，罪是会扩散的。

我们的盼望在哪里

如果一个完美的人性都不能胜过魔鬼的谎言，那么堕落之后，那破碎软弱的人性又如何能够胜过呢？面对圣经所指出的无所不在的“肉体的情欲、眼目的情欲、今生的骄傲”，面对今世无孔不入的“金钱、美色、权力、名声”的诱惑和欺骗，我们的又有多少胜算呢？没有，一丁点都没有。我们自以为逃脱了一个骗局，其实是又陷入另外一个骗局里。如果我们的人生就已经注定要落入一个罪的骗局里，那么我们的盼望在哪里呢？

笔者要说，盼望不在我们这里，而在神的儿子耶稣基督里。所有的人都在魔鬼的骗局中失败了，唯有耶稣得胜了，祂胜过了魔鬼的引诱，识破了魔鬼的谎言（太4章）。最后，祂在十字架上用死亡践踏了仇敌魔鬼的头，并且祂把祂的得胜赐给了凡信靠祂的人。

唯有真理能够戳穿谎言，唯有光能驱散黑暗。耶稣说：我是世界的光，跟从我的，就不在黑暗里走，必要得着生命的光。唯有来自神真理的言语，能使我们看清谎言的本质。

当然，我不是简单主义者，认为信了耶稣，人就变精明了，就变得灵巧有智慧了，就不会上当受骗了。基督徒受骗的不在少数，我自己信了耶稣之后，也照样上当受骗过。因为我们依然活在罪的阴影中，这个世界是罪恶掌权的世界，我们的内心虽然有耶稣的光，但依然有罪的影响，我们并不完全。所不同的是，因为真理的来到，谎言已经被戳穿，虽然继续存在，但已经不再有效力；就如我们所常听到的共产主义的谎言，它继续存在，但已经不再能呼风唤雨。基督徒虽然行在谎言的阴影中，但那阴影越来越淡，因为照着他的光越来越亮了。

就我来说，程序员曾是我的梦想，黑客就是我的偶像。因为我觉得代码几乎是万能的，可以帮你实现你想做的任何事情，只要会写代码就能把整个世界玩弄在股掌之间。2002年，我开始接触Java语言，开始进入网络和程序员世界的时候，我就写下两句豪言壮语：娶来java，嫁给net。只不过，我可能有掌控世界的野心，但我的人生却不在我的掌控之中。我刚跳入这代码的海洋中不久，就遇见了我的恩主耶稣，祂向我显明祂是神，我的救主和生命的主；在祂那里，我找到了人生真正的意义，非常幸福快乐。从此，一发不可收拾，就死心塌地的跟了耶稣，还成了一名牧师。我看牧师最重要的一份工作是：戳穿蒙蔽人心的谎言，让人重新活在真理中。在服事中，我看到一个谎言一次又一次的重现：去改变世界却不用改变自己。结果是：想改变世界的人，反而一次又一次的被世界所改变。其实，真理是：当你被改变的时候，世界就开始改变。

你也来逃离这场人生的骗局吧！
