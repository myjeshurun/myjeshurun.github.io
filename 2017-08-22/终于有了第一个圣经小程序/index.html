今日圣经小程序背后的故事（附攻略）

发表于 2017-08-22 | 分类于 工具控 | | 阅读次数

今日圣经小程序

近日，今日佳音的细心粉丝会在我们的公号上看到一个消息说，公众号关联小程序通知。如果你点进去，就会眼前一亮，看到一个神奇的新东西：今日圣经。

惊艳的那一刻，你会问，这是什么东东？

这是个小程序，可能有很多小伙伴们，还不知道小程序是什么，请看文后小编的介绍和短评1。

作为一个基督徒媒体机构，我们常常很努力的开动小脑筋，想如何使用新媒体把耶稣的宝贵福音更好的传扬出去，好和其他伙伴们一起完成大使命。这是我们最最开心的事！于是，我们一开始做广播，接着做网站；后来微博火了，我们就去做微博；智能手机很多人用，我们开发手机应用；大家都用微信，我们就经营微信公号：今日佳音wxbible；现在，小程序有人用，于是就有了：今日圣经小程序。哒哒哒…

所以啊，今日圣经小程序对很多人来说，貌似横空出世，其实并不是一个偶然。这是我们用心体会上帝爱世人，希望“人人得救，不愿一人沉沦”的心肠，并一直努力回应上帝在这个时代给我们呼召的结果。

小程序“怀孕”的那一刻

有一次跟安大（他是我们的老大，哈哈）一起出差，回来的时候，在飞机上他跟我倒起他对未来的忧虑。他诉苦说：某某机构做的非常优秀，我跟他们谈合作的事。我本想和他们一起鼓励信徒活出信仰，关注周遭的社会，并努力在现实生活中做好见证；但他们似乎不是感兴趣。神给了他们很大的祝福，为了获得他们的支持，我宁愿什么名分都不要，就把我们做的东西白白给他，不知道他们愿不愿意。我准备下次再找他们谈，但心里实在没底。

我呢，不小心也了解该他谈的这机构，就直接开导他说：哎呀，你也别难过。我看这个合作本来就很难成，因为神给他们的异象和托付跟我们很不一样。他们最看重的是神的圣言对信徒生命的重要意义，我们则比较理解现代人的习惯和节奏，看重信徒生命的更新，以及在现实社会中活出信仰，做好见证。我们本质上还是个媒体机构，跟他们不一样。你就别打这门子心思了，我觉得最好是我们想办法自己做。

不知道是我的话刺激了他，还是启迪了他，他突然瞪大了眼睛，对我说：“对了，小程序，就做一个圣经小程序，就叫“今日圣经”。然后，把我们现在所有的东西放在一起：每日灵修和佳音文章。小程序刚出来，会用的人少，但以后肯定会火起来。我们有王弟兄（我们团队中的技术牛人），他研究小程序有大半年了，他可以做。这是腾讯的野心布局，恰好给了我们机会。就这么办，哈哈…”

对对，可不能忘了，为我们做美工设计的春花姊妹，没有她，这小程序肯定不能长的这么漂亮…

说干就干，大家看到的小程序就是这样被造出来的，凝聚了一个团队的智慧和心血。愿荣耀归给我们在天上的神！

今日圣经的特点

  1. 功能简单，一站到位
    今日圣经小程序提供三个方面的功能和服务：阅读圣经、每日灵修和今日佳音，后两者的内容对应普世佳音做的两个公号：每日箴言和今日佳音wxbible。我们期待的是，这个小程序是大家平日喜欢用的，所以各方面都做得尽可能简洁，让大家用起来方便。
    圣经在线，就是提供给信徒一个在微信上读圣经的工具，加上搜索经文的功能；
    每日灵修，就是每日的灵修文章推送；
    今日佳音，每日一篇从信仰的角度回应时下的热点事件。
    我们想一个基督徒在忙碌的生活中，如果每天能做到读经、灵修，再了解一下当下的社会，就可以了。这就是我们做这个小程序一开始的初衷：为广大的基督徒提供圣经、灵修、生活三位一体，一站到位的服事。针对这三个功能，我们编了一个顺口溜：圣经就像手机在线一样时刻在我们心里，灵修是准时的赴上帝的邀约，佳音让我们关怀社会时事。
    我们期待，当弟兄姊妹的生命得到充实的喂养之后，他/她可以在他所生活的周遭世界中，为神做美好的见证。

  2. 界面简洁，专注阅读
    简单实用的功能，配上有简洁的页面排版，希望可以营造一种氛围，让大家可以享受阅读的过程。我们的期待是：不让微信公号上那些花哨的排版分散了我们的注意力，让我们可以更多体验去文字的神奇力量。

  3. 有声朗读，用心聆听
    声音，声音！我们为每一部分都提供了有声朗读，这样可以释放你的双手和眼睛，让你用耳朵用心来聆听神借着圣经、灵修小文和热点评析文章对你所要说的话。当你整天盯着电脑屏幕感觉眼睛很累，在地铁上、公交上不方便看手机或者开车在路上，相信这个有声的功能会让你很喜欢。

使用今日圣经的贴心小技巧

  1. 如何找到今日圣经小程序？
    打开微信，点击右上角的搜索，输入“今日圣经”，点击下方的“搜一搜 今日圣经”，在出来的搜索界面，你应该就会在最上方看到“今日圣经”小程序的图标，点击进入即可。
    或者呢，你可以打开微信直接扫描一下的二维码，直接进入小程序：
    （请其它小编在这里放上小程序的二维码）
    你点进去之后，建议你做两件事情，帮助你下一次快捷的找到今日圣经小程序：

    • 把今日圣经小程序放到微信聊天的顶部。同样的，在今日圣经小程序首页界面右上角点击“┇”，选择：“显示在聊天顶部”；这样下一次，你就打开微信跟朋友聊天的时候，或许想起还没有读经、灵修，就可以直接从聊天顶部点击进入今日圣经的小程序了。或许上帝知道我们的软弱，很难离开微信，因此就借着今日圣经在微信上遇见我们，对我们说话。

    • 把今日圣经小程序放到你的桌面上。在今日圣经小程序首页界面右上角点击“┇”，选择：“添加到桌面”；这样下一次，你就可以直接从你的手机桌面，点击进入今日圣经的小程序了。

      如果你上面两步都没有做，也不用慌；你只要在微信界面点击“发现”，找到下方的“小程序”，点击进入，你就会看到“今日圣经”小程序在那里等着你~~

  2. 如何推荐今日圣经给你的小伙伴？
    好东西要一起分享嘛，这是小时候爸爸妈妈教给小朋友的道理。
    • 分享这个小程序给别人，在小程序首页界面，右上角点击“┇”，选择“转发”，选择你要转发的人或者群就可以了。很遗憾，目前小程序还不能发布在朋友圈。
    • 分享小程序的文章给别人，在文章界面，右上角点击“┇”，选择“转发”，选择你要转发的人或者群就可以了。用这种方式不能分享到朋友圈。但下面介绍的另一种方式可以，请继续往下看。
  3. 【亮点】读（灵修）文章时随时回到圣经查阅。
    特别当你读每日灵修的时候，都会读到相关的经文。一般在文章中只会显示一节，但很多时候，我们想去查看圣经看其它的经节。一般情况下，你就只能退出灵修的文章，然后打开另一个圣经软件，比如微读什么的，然后找到经节出处，开始阅读。
    今日圣经在这一点上，提供了无与伦比的体验。当你点开一篇灵修文章，会看到开头的当日经文（只显示一节，底色灰色），你点一下经文，就会看到全部的经文；而你点击一下经文出处，就会带你到经文搜索，然后再一点击，就会看到一整章的经文。这样，会节省大家的时间，也让大家的灵修阅读更加流畅。怎么样，很酷吧~~
  4. 首页界面显示你最近的读经记录，当日的灵修文章，和当日的佳音头条文章。你可以点击进去阅读，也可以直接点击播放键收听。
  5. 点开声音后，可退到在微信聊天界面，会继续播放，不妨碍同时查看其它信息。这时，你在聊天顶部会看到一个播放信息，点击一下，你会有惊喜，或许还会惊艳哦:-) 还有更大的惊喜是，在这个界面，点击右上角的“┇”，不仅可以把这个有声文章分享给个人或者群组，还可以分享到朋友圈！

这只是一个开始，未来还有后续的功能更新，敬请祷告关注。若有想法、建议或者鼓励什么的，当然发现了什么Bug了，请在后台留言啊^_^
