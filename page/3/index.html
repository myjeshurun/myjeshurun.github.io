未命名

发表于 2017-04-17 | | 阅读次数

明星的高离婚率是一个不争的事实。但每当某个明星大腕爆出离婚或者外遇绯闻的时候，许多人还是会大跌眼镜，大吃一惊，觉得像他们这样有钱、有成就、有颜值的人，怎么也会婚姻不幸呢？这背后折射出来的观念，就是只要有钱、有成就、有颜值，婚姻就能稳固幸福。明星们婚姻一次又一次的破碎本足以打破这种看似合理却是错误的观念，但人们看见了却不领会，听到了也不明白，依然努力追逐明星们的足迹——有房有车有份稳定的工作。结局已经很明显了。
而不吃惊的人，大都是因为麻木了，见怪不怪了，觉得那些明星就是这副德性，一群只会演戏，“假冒为善”的家伙。他们还引经据典说，古罗马时代的人早就看清楚了，剧院的演员就是用“假冒为善”这个希腊文来称呼的。

其实，明星们的高离婚率并不奇怪，并不是这些明星本质上比一般人更坏，相反，他们是和我们有同样性情的人；他们跟一般人不一样的是，他们进入了一个叫“演艺圈”的怪圈，并且接受了这个圈内的游戏规则和价值观念，因而使得他们的婚姻比常人的不稳固。

我们每个人不一定要成为明星进入这个圈，才会受到这个圈的影响；而是，当我们接受了这个圈内的游戏规则和价值信念，那么我们就是圈内的人，我们的婚姻也是会像他们一样的脆弱。

当我们作为一个旁观者，一名观众去观察、解读剖析这个圈内的种种形态的时候，我们不是要去批判这些明星大腕的自私贪婪，这样做并没有什么益处；相反，我们看他们在本质上是和我们一样的人。幸福的家家相似，不幸的家庭各不同。在怜悯他们的同时，我们透过他们“不幸”的镜子来反观我们自己，期盼我们可以避免他们的“不幸”，获得渴望的“幸福”。

  1. 职业与婚姻
    凤凰网报道，国外媒体对美国人口普查所做出的调查显示，从事艺人或运动员这个职业的人，离婚的机率比其他职业的人更高，原因很简单：工作时间不固定，而且生活的环境中诱惑也更多，让另一半没有安全感，夫妻双方容易产生争执。
    我们一般对明星艺人都是羡慕有加，但他们光鲜亮丽、鲜花掌声的演出背后却也承受常人所没有的痛苦和挣扎。他们也是人，不可能一直在工作（就是演戏），他们也要生活。对大部分演员来说，演戏并不是他们的信仰，只是谋生的一个手段，或者更深入的一点说，实现自己的人生价值。我们要求这些艺人在现实生活中活出所演绎的荧屏形象，就是要求他们把工作和生活合二为一，从一般的情理上讲，这是对他们过高的要求。不过，对艺人来说，要是他们能够在现实生活中能“入戏”一些，他们的婚姻或能更长久。
    在罗马时代，有个很有名的犹太思想家，他的作品至今被许多人天天诵读、默想；他的思想启发了500年前马丁路德的宗教改革，从而深深的影响近现代的各种思潮。人们称呼他叫使徒保罗，他说自己的人生就像“一台戏，演给世人和天使看”。我们也常说，人生就像一个舞台，我们都是舞台上的主角。其实，演戏也是一件很严肃的事情；不过，这场戏是我们的整个人生。

  2. 婚姻不是一纸工作协议
    对艺人的职业生涯来说，工作协议是很重要的。从一个东家到另外一个东家，从一场戏到另外一场戏，他们是按着协议来工作的。协议是一个双方商讨达成的约定，里面规定了双方的责任和收益。这是一种契约，基础是双方的利益；一旦这个利益的基础被打破，契约很快就会被撕毁。这是商业社会的游戏法则。
    但问题是，当人们也按着这个世界的游戏法则来对待婚姻的话，婚姻就会像纸上的合同契约一样容易被撕碎。有人为了保护自己的利益，就事先在这份契约上清楚明了的指出：如果离婚，财产如何分配；儿女如何归属等等问题。
    这是软弱的人面对软弱的自己和软弱的配偶无奈之下的一个选择，但这样的婚姻看起来稳妥万无一失，并不能让人满足，更不能让人幸福。因为真正的婚姻自古以来就不是一纸合同，而是一个以身相许的承诺，一份生死与共的盟约；婚姻就是这样被设计的。古老的圣经就在开始的篇章中，写下人类婚姻的精髓：因此，人要离开父母，与妻子联合，二人成为一体（创世记2:24）。在基督教的婚礼上，在牧师庄严宣告新郎新娘二人成为夫妻之后，都要郑重的加上一句：神所配合的，人不可分开。
    这一方面是回归婚姻本源的设计，也是对人的软弱做出一个回应。因为人的本性就是容易受诱惑而轻言放弃，当离婚的后路被切断，只剩下面对婚姻问题的时候，人反而能在破釜沉舟的绝地中找到一条出路，使婚姻重获新生。

  3. 真正的幸福不是占有
    大多数的爱情片和情歌演绎的都是“得到和失去”的故事和情感，其中所宣扬的哲学就是得到就是幸福，失去就是痛苦。这是人性很真实的一个层面，但是比较表面、肤浅的层面。如果用生命成长的过程来衡量的话，这个阶段好比孩子对世界的理解和把握。我并不是说孩子的理解就是不可取的，是没有价值的，这是生命成长的一个阶段，同时也呈现生命的一部分真理，但这不是全部，人也需要不断的成长。可惜的是，我们的媒体充斥了大量这种东西，而参与创作这些东西的艺人也在其中获得成就感和价值感，以为这就是生命的全部；直到现实的婚姻把这座无比绚丽却是纸搭的城堡推倒。
    如果艺人们能超越他们在工作中所演的戏剧，透过婚姻看到生命本身上演的更广阔、神奇的剧目，他们的婚姻会不一样。有人说，人生就是一个悖论。这句话很有道理，婚姻的成功法则不是你承担50%，我承担50%，而是我承担100%，你承担100%。婚姻幸福的秘诀不是通过占有来获得自己的满足和幸福，而是在给予和付出中满足对方，让对方得到幸福。正如耶稣所说的：施比受更为有福，真正的快乐是在给予和失去中找到的；这不是一个玄妙的真理，而是一个在现实的生活中可以经历和活出来的真实。

  4. 婚姻不是两个人的事。
    明星艺人受人瞩目的职业特点，决定他们的婚姻是一座孤岛。但一个稳固的婚姻需要长辈的引导，需要团契的支持。婚姻是两个人的事，但又不是两个人的事。没有一个世代像我们这个时代，人们都带着面具生活，造成人与人之间有种很深的隔阂。朋友圈一个个的分享、一张张的照片给人一种假象说：全世界其他人都过得很幸福，只有我们的婚姻很糟糕；殊不知那只是生活的一个个截取的片段，一个个的时刻Moment，并不是真实的生活本身。演艺圈更是强化了这个粉饰的现实：有哪一个演员上台不化妆不穿戏服的？只有展示最亮丽照人的那一面，只有带着面具才能赢得眼球，获得掌声。于是，婚姻里面那真实的不堪自我是不能为外人道，只能在黑暗中隐藏的。而婚姻是经不起黑暗吞噬的。
    在我以前所牧养的教会中，只有几对夫妻。我们就开始了夫妻团契。当有人开始分享家庭中那些烦人的琐事，夫妻间的不合、挣扎和痛苦，别人就会回应说：啊，你们也是这样啊。我以为只有我们的婚姻这么糟糕。有意思的是，大家分享完自己家的“丑事”特别释放，加上一起祷告交托给上帝之后，更觉得特别轻松，好像身上的重担脱落了。因为我们深深感觉到，我们并不孤单，并不是只有我们在经历破碎；这种有同伴一起经历生命破碎的感受，给了我们很大的力量。很多时候，婚姻的重担不仅来自夫妻二人，更是来自外界无形的压力。我和妻子至今还很怀念那段美好的时光。

####结论：留心职业对婚姻的影响
以上只是列举的演艺圈艺人婚姻的一些特点，当然还有更多，读者可以自行去观察、揣摩。我想我已经说明了，艺人的职业本身很深的影响了他们的婚姻。这一点可以给我们一些启迪，因为我们也都有各自的职业。
职业本身影响我们的婚姻。这是可以理解的，因为每个行业都有它特有的模式和法则，我们人都不是割裂的，职场会塑造我们。这种塑造有好有坏。好的我就不说了，只说说不好的。
当我们听到人说，不要把工作带到家里。这是一个很好的原则。但我们不能认为，只要不把办公室的事务带到家里就可以了。其实，这远远不够；因为职场上一些规则、态度、做法并不适用于家庭生活；而这些却是潜移默化的进入到我们心里，影响我们的。举个极端的例子：一个在公司里做老板管着手下许多男性员工的女人，如果在家也是这样命令她的丈夫，那么丈夫就会失去男人在家里应有的尊严和担当；夫妻关系很可能因此受到伤损。

很少人可以为着婚姻而丢弃工作，但同样我们也不愿意为了工作而毁了婚姻。在工作和婚姻划一道清楚的界限是很有必要的。当你回家脱掉工作服、换上宽松的家居服的时候，同时提醒自己：脱掉工作中特有的态度和习惯，比如：决断、命令、事情导向等，穿上属于家庭生活该有的爱、温柔、自由、嬉戏等。

####进一步的思想
你可能会发现，这些特点虽然与他们的职业紧密联系在一起，却并不是他们所独有的。造成他们许多人婚姻不幸的问题，也发生在我们每一个人的身上。托尔斯泰说，“幸福的家庭家相似，不幸的家庭各不同”，也不尽对，不幸的家庭其实也有很多相似之处。如果我们忽略工作对婚姻不好的影响，看婚姻只是一纸协议，渴望在占有中得到幸福，并且没有朋友的鼓励和支持，我们的婚姻也会像“演艺圈”内的婚姻一样，处在极大的风险之中。但我们若能有意识的调整自己的生活和态度，就能使自己的婚姻稳固一些。

最后，我要以之前提到的使徒保罗的话来结束。他之前说到，他的人生成了一台戏，演给世人和天使看。他后来提到，他是如何演好人生的这场戏的。他的答案就一个字：爱。我想，保罗所说的爱也可以回应我们上面所说的问题。不过，首先要说明的是，这个爱适用于世上一切人与人之间的关系，要远远超越爱情和婚姻的范畴。但因为极其切合人们所谈论的婚姻之中的“爱”，所以自古以来就是基督徒婚礼中被引用最多的话，用以勉励所有将要进入婚姻和已经在婚姻中的男女：

爱是恒久忍耐，又有恩慈。爱是不嫉妒，不自夸，不张狂；
不作失礼的事，不求自己的益处，不轻易动怒，不计较人的过犯；
不喜欢不义，只喜欢真理。
爱是凡事包容，凡事相信，凡事盼望，凡事忍耐。
爱是永存不息的。
——哥林多前书13章4-8节

未命名

发表于 2017-04-11 | | 阅读次数

现代很多人无法接受耶稣复活，觉得这是神话，没有科学根据；基督教信仰建立这基础上，因此很不可信。记得我大三刚信主的时候，我跟父亲讲福音，提到耶稣从死里复活。他一听，就跳起来，好像神经受了很大的刺激，指着我的鼻子，咬牙切齿的骂我说：“看来我白供你读书了，你书都读到牛背上去了（闽南俗语，意思是：书都白读了）。现在人一死，就到火葬场去火化，一下子就变成灰了，哪里有可能复活呢？”我父亲是中国南方的精明的小生意人，文化不高；讲求实际，非常勤劳，崇尚个人奋斗。他的至理名言就是国际歌的词：从来就没有什么救世主，也不靠神仙皇帝。在利益当头，他是天生的唯物论主义者，但这不妨碍他去烧香拜佛求个平安。

面对父亲的问题，我当时也没怎么回答他，也知道跟他说不通；对一个只相信眼见不相信神迹的人来说，死里复活怎么说也是不可能的。 对朴实的唯物主义者来说，眼见就是科学；眼睛不能看见的，都是虚的，都是迷信。他们认为自己受教育程度不高，烧香拜佛也就算了；对于受过高等教育的人，去相信一个所谓的“死而复活”的神迹，简直是天方夜谭，天大的笑话。上周主日，一个姊妹分享到，她的学化学的同学就是如何也无法接受死人能复活的道理，特别是在执行火葬的时代。

那么，基督徒传扬耶稣从死里复活了，并宣称这是确切的真理。这是否是伪科学或反科学呢？耶稣从死里复活有多少的可信度呢？

#####耶稣复活是哪一门子的真理？

  1. 耶稣的复活是历史的事实
    首先要区分两种真理，或者叫事实：一种是科学实验可以证明的真理，比如：万有引力定律，化学反应原理等；另一种是历史上发生的事实，比如抗日战争，第二次世界大战等。许多人认为耶稣的复活不符合科学，这是犯了类别错误；因为耶稣的复活属于历史的真理，而不是科学的真理。

历史的真理是由历史的证据来确立的，而不是由科学实验来证实的。历史的证据有人证、物证。

最大的物证就是耶稣的空坟墓。耶稣在被钉十字架后第三天复活，根据圣经的记载，他们的门徒在50天后（正好碰上当时犹太人的三大节期之一，有点中国的春节，犹太人从各个地方回到他们的圣城耶路撒冷）就开始大力传扬耶稣死里复活的消息。他们告诉人说：埋葬耶稣的坟墓是空的，耶稣的尸体不见了，耶稣复活了。如果门徒所传的不是真，这是一个相当容易被人捅破的；因为人们，特别是那些反对耶稣门徒的宗教领袖只要反驳说，看啊，耶稣的坟墓在那，这些人睁着眼睛说瞎话。结果那些最有可能揭穿背后真相的那些宗教领袖，只编了一个蹩脚的解释，他们教看守坟墓的士兵这样说：‘他的门徒晚上来了，趁我们睡着的时候，把他偷走了（马太福音28:13）。看来这些士兵都有特异功能，睡觉的时候也能清楚察觉到周围的动静。

最大的人证是耶稣的门徒。有人认为耶稣的门徒跟耶稣的关系太密切，所以他们的见证会有失偏颇，不客观；但其实恰恰是因为他们与耶稣的亲密关系，才使得他们对耶稣的认识更准确、真实，有说服力。这好比最了解北京的是土生土长的北京人，而不是上海人；最了解男人的是他的女人，而不是他的朋友。耶稣的门徒并不是一群杰出高尚的人；相反，他们是普通老百姓，文化不高，悟性也不高，粗鲁的粗鲁，火爆的火爆，多疑的多疑，怕死的怕死，还不怎么讲义气。12个人中有一个出卖了耶稣，另一个三次否认自己认识耶稣，其余的呢，在耶稣被抓的时候都跑路了。就是这样的一群小民，经历了奇异的改变，在耶稣死后，带着极强的使命感，大胆传扬耶稣复活的好消息，叫人悔改归信；他们不再惧怕死亡，并且绝大部分为他们所传讲的耶稣死而复活的福音付上了生命的代价，流放的流放，上刑的上刑，吊死的吊死。据说，那个三次否认耶稣的门徒——他叫彼得，觉得自己很羞愧，要求自己倒钉十字架而殉道。耶稣的这些门徒生命的改变，最合理、自然也是最好的解释就是，耶稣的复活，就如他们所见证、所传扬的。

另外一个耶稣的门徒叫保罗，他是基督教发展史上最关键的人物之一。他原来是非常的仇恨基督徒，不是因为他本性邪恶，而是他属于那种宗教极端热心分子，有些类似现在的ISIS成员。他后来成了忠心跟随耶稣的仆人，30年时间里传道的足迹横跨亚、非、欧大三洲，奠定了基督教信仰在希腊外邦世界的根基。除了传道宣教的工作，他还写了新约最早的13卷书（总共27卷）。原来逼迫拆毁教会的，现在成了教会的柱石；为什么会有这样大的反差呢？保罗他自己做见证说，促成这个改变的是复活的耶稣向他显现。
在他的其中一封写给教会的信中（大概是主后55年），面对教会中有人怀疑复活的事，他列出见过耶稣复活之人的名单：耶稣的十二个门徒，耶稣的兄弟雅各，还有五百多弟兄。然后，保罗直接告诉他们说：这些人好多都还在，不信的话，去问问他们（林前15:6）。

耶稣的死而复活除了圣经记载之外，还有其它经外文献有提及，最有名也
最权威的犹太史家是约瑟夫（他并不是基督徒）在他的《犹太古史记》中（此书完成于耶稣被钉十字架63年），这样描述当时的情况：耶稣…行了许多神迹奇事，也是一位让人们心悦诚服接受教诲的夫子。他身边有许多跟随他的犹太人和外邦人，这人就是“基督”。彼拉多在我们领袖的唆使下，将他判处十字架的刑罚；那些爱他的人从一开始就没有舍弃他，因他三天后复活并对他们显现，就如同先知们早已预言这件事及其他数以万计的有关他的事迹一样；因他而被命名为“基督徒”的一派，至今仍然没有消失。（第18卷第3章，粗体为笔者所加）

有本书叫《新铁证待判》（宗教文化出版社，2006）详细列举了支持圣经故事和耶稣真实性的证据，想更多了解的话，可以去仔细阅读。

  1. 普世基督教会的生命见证
    之前提到过，耶稣被抓的时候，门徒四散，死的时候只有他一人孤零零被挂在十字架上。从现代人的角度来看，耶稣的人生是失败的，他三年半的传道生涯，虽然有许多人跟随（或许准确的讲，应该叫“尾随”），但他并没有建立什么教派、党派、军队或者帝国。某种程度上，他最终因言获罪，耻辱的死在十字架上；跟随他的人四散。跟随一个刚刚萌芽，而教主就被钉在十字架上的教派有什么前途呢？奇特的是，就如约瑟夫所见证的，这一派“基督徒”（后来的人给耶稣的跟随者的一个称号），至今没有消失。不仅没有消失，而且越发壮大，大约300年后，从一个巴勒斯坦小镇开始的基督教居然成了当时整个罗马帝国的国教。要知道，当时教会就像刚刚出生的婴儿，而罗马帝国好几次残酷的压迫——抓捕传道人、将基督徒仍进斗兽场、钉在十字架上等，竭力要扼杀这个婴孩。但好像有一种超自然的能力，保守托住并引到这年轻、软弱的生命，就如耶稣自己说：你在婴孩和吃奶的口中建立了赞美（太21:16）；更像保罗所说：我们有这宝贝在瓦器里，是要显明这极大的能力是属于　神，不是出于我们（林前4:7）。

教会这个小幼苗，历经各种风雨磨难，如今已然长成参天大树，在她的荫凉下的来自各样的民族、国家、方言。距离耶稣复活大概2000年，基督教已经是全球最大的宗教，耶稣的教会更成了世界上最大、历史最悠久的组织。更难能可贵的是，虽然敌对她的外在环境没有多少的改变，上个主日发生在埃及的两个教堂爆炸事件造成上百个信徒伤亡就是一个明证，但她依然持守、并宣扬2000年前的信徒所相信的：耶稣死了，第三天复活了。

这一切是如何成就的呢？无他，耶稣的复活是也。

#####中国教会的真实故事
不仅全世界2000年耶稣基督的教会的成长真实见证了耶稣的复活，中国教会的200多年的发展也是一个明证。虽然耶稣的福音最早在唐朝就传入中国，但真正在中国这片土地上扎根要从1807年，新教的传教士马礼逊进入中国开始。成千上万的西方宣教士听见耶稣的呼召，放弃舒适的生活条件、锦绣的生命前程，来到一个陌生又不友善的国家，经过100年左右的耕耘，教会在1900年义和团运动的时候，大概有了7万左右的信徒。义和团运动却摧毁了大概3分之一的果子，包括2万的信徒和上百个西方宣教士。

经历这场血与火的洗礼，西方宣教士不仅没有退缩，反而更多的进入中国。他们其中最有名的代表是来自英国的戴德生，他1854年22岁到达中国，深入到中国偏远的内地，提倡接受认同中国人。因此，他穿中国人的衣服，留长辫子，做一个地道的中国人。不仅他自己把生命献给了中国，他的子孙后代继承他的心志，继续服事中国人，如今已是第6代。不用说外国人，就是我们自己，子孙6代做同样的一件事情的也是少之又少。戴德生有一句至理名言，如今依然感动、激励许多人：“我若有千磅英金，中国可以全数支取；我若有千条性命，中国应该拥有他们。不，不是为中国，是为基督。”

人们感概一位外国人竟然有对中国人这么深的爱，但戴德生爱的源泉不是一种精神，一个信念，乃是一位活着的主，就是耶稣基督。正因为耶稣是活着的主，才有后来中国教会在经历了1949-1979年更大、更长的逼迫和患难之后，依然存活并且发展。在那个什么主义、思想、宗教被禁止，教堂被封、圣经被焚毁，牧师被关进监狱、西方的宣教士都驱逐出境的“一清二白”的时代，基督的教会在中国就像种子埋在地里，以一种奇特而神秘的方式在生长着。从1807年——1949年，西方约2万名的宣教士到达中国这片福音的荒野开垦，历经140年，收获了不到100万的信徒；但1949年——2009年，50年的时间里，中国教会经历腥风血雨的政治洗礼和许多的逼迫打压，涨到至少5000万，有50倍甚至更多的增长。这是全世界教会发展中都是一个奇迹，唯一合理的解释是耶稣复活了，祂在掌权。

#####信徒复活的见证
世界上没有一宗历史事件，像耶稣复活这样，经得起两千年来无以数计的哲学、宗教、法律及历史学者的攻击和考证。而基督教会整体的生命成长是耶稣复活的最有力证据，这整个大拼图的背后是千千万万信徒个体生命经历耶稣的复活的大能，生命发生翻转的小拼图。而最近在美国上映的纪录片《重审基督》就是这样一个故事——

作者史特博是一位美国记者，自称无神论者。之前曾任职美国著名日报《芝加哥论坛报》(Chicago Tribune)，擅采访重大法庭新闻，屡获新闻奖。在写《重审耶稣》前，他极其藐视和反对基督信仰，相信有更多证据显示所谓的“神”不过是人心理投射，是古代神话、古人迷信；而耶稣所自诩的事，亦属杜撰。
后来，因为妻子信主生命发生改变，史特博用了两年时间访问在剑桥、普林斯顿等学府的十三位著名学者，盘根问底，再三提出十分尖锐的问题，诸如圣经新约的可靠性，圣经之外有无其他支持耶稣生平的证据，复活是否真实历史事件等。他原想推翻基督的证据，但他的问题反过来都得到完满解答，而且结论完全出乎意料。最后他自己也归信了基督，他的书《重审基督》竟成了为基督辩护之作。

#####笔者的生命故事
笔者最终成为基督徒，并没有什么经历曲折婉转、惊天动地的心路历程，只不过是听到了耶稣那温柔而亲切的邀请。在笔者受朋友影响，开始想去寻求神的时候，第二次进到教堂听道，台上的老牧师讲到耶稣的话说，“看哪！我站在门外敲门；如果有人听见我的声音就开门的，我要进到他那里去，我要跟他在一起，他也要跟我在一起吃饭。（启示录3:20）”这话深深的抓住了我，我闭上眼睛，仿佛看到耶稣站在我的心门外，轻轻的叩着门，那么轻柔那么深情；还不知道祷告是什么的我，就在心里默默的祷告说：”主啊，是你在我门外叩门，我愿意打开门，请你进来。”从此之后，我成了一位基督徒，后来也成了一位传讲耶稣死而复活的牧师。

而一开始提到笔者的父亲激烈的反对基督信仰，嗤之以鼻，但这么多年走过来，看到耶稣在我们一家中真实的恩典和赐福，他也慢慢平静下来，不再反对我们。虽然他还没有走进信仰，但也实在看到耶稣不是假的，而是真实的。

#####自然界到处有复活的见证
可见，耶稣的死而复活不是一个能用科学实验考察的道理，更不是神话，或某个人编出来的故事，而是真实的历史事实，并且是2000年每一个基督徒的生命检验过的可靠的真理。不过，虽然复活不是属于科学的真理，但是自然界也不缺乏这种死而复活的见证，我要说到两个典型的例子。

  1. 南方的农民都种过水稻。每年春天到了播种的季节，父母都会精心挑选好水稻的种子，先是放在一个大盆子里让它们长出牙；然后再把这些种子很密集的撒到准备好的一块平整的小田地上，再用一大张张开的塑料纸盖上。过了一段时间，这些种子就会长出一两寸高幼嫩的秧苗。把塑料袋拿开，再等一段时间让秧苗再长高一些；到了插秧时节的时候，他们就把长好的秧苗插到水田里。到了夏天收割的时候，这些秧苗就会长成一片片金黄的水稻。
    这些水稻种子没有被吃掉，而是被埋在地里；某种程度上，你可以说，他们死了；但它们又活了，它们长出了更多的水稻种子。耶稣在圣经中也说过类似的比喻，祂用的是麦子的比方：一粒麦子若不落在地里死了，仍旧是一粒；如果死了，就结出许多果实来。
  1. 常给孩子读绘本的父母，自然不会错过艾瑞·卡尔（Eric Carle）经典的《很饿很饿的毛毛虫》，里面讲述一只毛毛虫从小变大，最后结茧变成美丽的蝴蝶的故事。那种胖胖的、丑丑的、活在二维空间的毛毛虫死去了，蜕变成了轻盈美丽、绚丽多姿、在广袤的世界里自由翱翔的蝴蝶。这正是基督徒生命的隐喻，当生命中那些败坏的性情、欲望不断死去的时候，一个良善圣洁充满爱的新生命正在成长。

当圣经描述信徒最后身体复活的时候，用的正是这两个自然界的原型：我们现在世界的这个身体，正是未来世界那个身体的种子；这个身体是软弱的、会朽坏的，它会疲倦，而且终有一天要衰老、死去、腐烂；但复活的时候，却要长成另外的身体，是不会朽坏的。这有点类似：一粒水稻的种子长成稻穗，一颗树的种子长成大树；又像令人发毛的毛毛虫变成美丽轻盈、展翼招展的蝴蝶。

在这个意义上，耶稣的死而复活可以看做是整个世界生命延续的奥秘在人类肉身世界中的历史彰显。基督好比一粒种子落在地里死了，许许多多的像基督一样的生命诞生了；一个圣徒死了，许许多多的像圣徒一样的生命诞生了。基督复活了，祂复活的生命进到每一个跟随祂的信徒的生命中成了一粒种子。这就是基督徒生命的故事——一个个死而复活的传奇。

#####结尾

相信耶稣从死里复活这个事实并不难，这某种程度上就相当于你相信了地球是圆的，而不是像古人所认为的方的。关键在于，耶稣死而复活的意义。整个基督教信仰都围绕这个展开，这是基督教信仰的核心；可以说，整本圣经洋洋洒洒90多万字都在谈论这件事。我在这里，就说其中一个意义：耶稣胜过了死亡这个人类最大的仇敌，祂是生命的主，祂是复活的主。一个很有名的传道人曾经说过：人类的历史无非就是坟墓的历史。再厉害的人能征服全世界，却最终都要向死亡低头。如果耶稣胜过了我们最大的仇敌，那么其他的仇敌更不在话下了。圣经说，耶稣死，是为我们的罪了；祂复活，是为我们称义，叫我们有永生的盼望。

亲爱的朋友，这就是基督徒所说的福音：耶稣为你的罪被钉在十字架上，死了；第三天复活。今天，这位活着的耶稣依然透过圣经发出充满爱的邀请说：”到我这里来吧，你们所有劳苦担重担的人哪，我必使你们得安息。我心里柔和谦卑，你们应当负我的轭，向我学习，你们就必得着心灵的安息；我的轭是容易负的，我的担子是轻省的。”（马太福音11:28-30）

当你听见这福音，并愿意悔改自己罪的时候，这位死而复活的耶稣就会进到你的里面，赐给你永恒的生命。

####孤独的王林
王林是可怜的，是孤独的；因为只有他自己深深知道自己的底细：他不过是一个有着血肉之躯的软弱之人，有着和正常人一样的七情六欲，他和正常人一样需要被医治（肉体的和心灵的），需要被救赎；却不得不扮演一个属神的角色，给别人医治，给别人拯救。最后，只有让死亡来显明一切真相。无论他的死是由于自然的生命法则还是人为的权力法则，终究他死了；既然他不能救自己脱离终极的死亡，那么之前所有的“拯救”就都是短暂的，也是无效的。

其实对一些很属灵的基督徒，还有牧师来说，不也常常面临着同样的试探吗？当人们都来仰视你，崇拜你，急切的需要你为他们提供特别的属天灵粮的时候，你觉得自己跟他们不一样，你觉得自己是他们的“救星”，你觉得自己不能软弱，不然他们就要跌倒了；因此你就隐藏你自己真实的挣扎，真实的软弱，甚至真实的罪。最后的结局往往是可怕的。圣经说我们不过是罪人一个，即使在基督里蒙了重生，得了救赎，最终也永远只是人。这意味着，我们的终极存在不是一个个只需要神同在的属灵人，而是。神给我们的诫命是去给予爱，去爱神爱人，这是一条永恒的诫命；但许多人忽略了这条诫命一个隐藏的前提是：我们是接受爱的人：被神爱，也要被人爱。其实，只有一个向神向别人敞开，懂得接受神的爱，也懂得接受别人的爱的人才能真正的去爱神和爱人，也才是一个真正的人。神把我们造成人，也呼召我们作为真正的人活着！这样的人，具有纵向和横向的生命维度，就如十字架的两块木板：纵向对那位创造的神有敬拜感恩又依靠接受祂的恩典；横向对身边的邻舍敞开接受又慷慨给予。

####悲剧的王林
毫无疑问，王林的人生是场悲剧，就如圣经所说，他死在了自己的罪中。他是一个被自己贪婪的罪深深辖制的人。他结交名人显要，哄抬自己的身价，如果报道属实，那么他向他的徒弟邹勇索要740万的拜师费简直让人瞠目结舌；此后和邹勇闹翻，杀人灭口实在也是狂妄自大到了极点。后来，虽然被新京报、人民日报和央视曝光内幕，还是不肯收敛，依然故我。正应了圣经的话说：贪财是万恶之根。有人贪爱钱财…就用许多痛苦把自己刺透了。（提前6：10）。

但没有一个人是一座孤岛，他的悲剧也毫无疑问是我们这个时代造成的，是我们这个弯曲悖谬的世代孕育了这样一个生命的怪胎。我们很容易把矛头指向那些利欲熏心的贪腐官员和许多爱慕虚荣的明星大腕；的确是他们直接把这样一个人“伪气功师”捧上神坛。但如果这些人是脐带的话，羊水就是我们这些芸芸众生。

####谁来帮助王林？
我在网上的报道中发现，王林的一生中做了不少慈善，捐了很多钱给穷人。在2015年的一次捐赠中，他说，小时候是“苦儿子”，也见到很多老人老无所依，逢年过节只能闻别家的鱼肉香，没少心酸。他说至今已坚持捐款20多年了，最近几年每年都捐100卡车物资，希望让“最苦的人也能过好每一天”。无论他真正的动机如何，事实是不可否认的：他坚持做慈善，确实的帮助了不少人，不少地方。耶稣说：你们所做的，只要是作在我一个最小的弟兄身上，就是做在我身上了。（太25:40）教会历史上，常用耶稣的这段话来作为教会做慈善关怀穷人的经文支持。而耶稣说这话不仅是对教会说的，而是对万民万族说的（太25:32）；而且耶稣在这里也并没有提到人们行善的动机如何。既然连耶稣都记念这些善事，我们不能因为否认。这说明，他的内心里面还是有好的良善的部分。可惜的是，这些好的、良善的部分并没有被发扬、被保存，而是被其它的恶所淹没、所吞噬了。我在想，这些领受恩惠的人中有没有基督徒，他们有没有因此感谢神并为他这个有罪的人在神面前祈求祷告？而领受王林恩惠芦溪当地人或者其他地方的人中有没有人找机会真诚的向他表达过真诚的爱戴感谢（而不去顾虑他所做的其它事，因为那是别人管的，或者上帝管的），感谢他的捐赠是如何祝福了他们的家庭和孩子，如何影响他们的人生？每一个人不管穷人还是富人，他都会因为能帮助别人、祝福别人而感到有意义有价值的。而这种价值感和意义感可能会改变一个人！

面对一个贫富悬殊的东西方世界，特蕾莎修女深深相信穷人和富人是彼此需要的。穷人需要富人的帮助，是显而易见；但富人也需要穷人。她曾经说过：“比物质贫穷更可怕的，是精神的荒漠；不仅穷人需要爱，富人也需要心灵的甘泉。”看看最近媒体报道的许多中国富人的故事，这些话他们来说是再切合不过了。穷人的内心常常所拥有的知足、感恩和喜乐，往往是富人所需要而没有的。

####普通人能做什么？
作为一个普通人，特别是一个有责任感的普通人，活在这个世界的确很容易觉得沮丧失望，因为问题看起来是那么的巨大，而我们的力量又是那么的渺小。面对挑战，我们简直就像螳臂挡车，杯水车薪。但上帝没有让我们去改变这个世界，至少没有让我们靠着自己的力量去改变；祂呼召我们天天来悔改自己的罪，信靠跟随祂的儿子耶稣。如果我们想改变这个世界，首先要来改变自己；而改变自己是最有可能改变别人，改变这个世界的路。Think BIG，but do SMALL. 对普通人来说，如果我们能靠着那加给我们力量的主耶稣，胜过我们里面对别人成功、富有、地位、才干的嫉妒，不去羡慕别人的拥有，满足于上帝所赐予我的所有；在贫乏和不足中也能坦然去接受别人的帮助，并喜乐感恩；而在自己的富足中，又能慷慨去分享给那些比自己缺乏的人。这样的生命，这样的爱，会被上帝记念和使用，带给这个世界更新和祝福。

美国著名作家Frederic Buechner写过这样一段话，我觉得是除了圣经·哥林多前书13章论到爱的段落之外，关于爱最美、最经典的描述。谨以这段作为本文的结束：
爱那些与你一样平等的人是人之常情——那是朋友对朋友的爱，弟兄对弟兄的爱。这是对那些爱你之人和那些可爱之人的爱，世界因此而微笑。

爱那些不幸的人是一件很美的事，这是对那些受苦的人、贫穷的人、生病的人、失败的人和不可爱之人的爱。这是一种同情，这种爱触摸到世界那脆弱的心。

最后，有一种爱，是对仇敌的爱——爱那些不爱你反而嘲弄你，威胁你使你痛苦的人。这是受害者对施暴者的爱。这是神的爱，正是这种爱征服了整个世界！

对中国人来说，今年是“鸡年”。有些基督徒觉得这种叫法很不妥。“鸡年”还好一些，但再过7年就是“龙年”；龙在圣经中，是象征魔鬼撒旦的。把某一年标记为撒旦的年份，这怎么也不能让基督徒接受啊了。甚至有人觉得，承认十二生肖就是偶像崇拜，是基督徒应该弃绝的。

这的确是中国基督徒难以回避的文化课题。我们一方面要持守圣经的原则，不沾染偶像的污秽，神所不喜悦的事，我们要靠着神的恩典去远离；另一方面又要按着圣经所教导的“若是能行，总要尽力与众人和睦”（罗12：18），因此我们可以向不信的人显示出上帝的美善和温柔。这篇文章主要回答一个问题：生肖与偶像崇拜到底是什么关系？然后简单探讨一下基督徒该如何本着圣经的原则去回应这个问题。

#####生肖的起源与来历
关于十二生肖的起源，或学者的考证、或民间的传说，众说纷纭。跟动物崇拜是否有关系，各人的见解也是不同。有人认为，生肖起源于原始时代的动物崇拜，十二生肖对应着十二神兽，用来驱除四方疫鬼。也有人认为，十二生肖跟中国干支纪年是连在一起的，十二种动物是十二地支的一一对应，以动物作地支标志。用来记人的出生年的十二种动物，即鼠，牛，虎，兔，龙，蛇，马，羊，猴，鸡，狗，猪。如子年生的人属鼠，丑年生的人属牛等，这就是我们常说的十二属、十二属相、十二生肖。因此，在古代的中国，有了这个十二生肖，贫民百姓就能记住自己出生的年号。这种简单的动物纪年法，后来称其为“生肖年”（生肖：“生”出生；“肖”相似、相像）。

不管怎样，大量的文献资料证明，生肖的确起源于中国，是华夏先民动物崇拜、图腾崇拜以及早期天文学的结晶。我想这个结论是蛮合理的。这也印证许多基督徒内心的忧虑，生肖属相的习俗源于偶像崇拜，是否该彻底摈弃？不然，我们不也成了偶像崇拜了吗？

在我们进入这个问题之前，先谈谈圣经说到的偶像崇拜是怎样的。

#####什么是偶像崇拜？
圣经向我们清楚的启示：有一位独一的真神，祂创造了天地和其中所充满的一切，包括人和各种动物；并且祂在基督耶稣里救赎了我们这些罪人脱离罪恶和死亡的辖制，使我们成为祂的儿女，赐给我们永恒的生命。
作为被造的人，我们承认：神是创造万物的主，是一切生命的源头，并且祂是供应生命、维系生命存在的那一位。
作为被救赎的人，我们承认：神是我们唯一的依靠，唯一的拯救。
正是因为有这个独一真神的信仰作为根基，我们才能分辨什么是偶像，没有真理就无所谓谬误。神的独一性天然的排除了其他偶像的正当性，就如十诫的第一诫，神清楚的宣告：除了我以外，你不可有别的神。对神的百姓来说，这不仅是第一、首要的诫命；也是神的子民对自己最基本、最关键的身份认定：我的存在是因为这位独一真神的存在而有的。

如果神是创造主、救赎主，祂是我们生命的源头、依靠和盼望，其他所有一切不是神的东西却被当做的生命的源头、依靠和盼望，就都是假神、偶像。如何分辨偶像呢？他们“是人手所造的，有口却不能言，有眼却不能看，有耳却不能听，有鼻却不能闻，有手却不能摸，有脚却不能走，有喉咙也不能出声。”（诗115:4-7）一句话：偶像本身是没有生命的，不能给予生命，更不能救赎生命。而真神是生命本身，祂创造生命、维系生命，也救赎更新生命。

偶像崇拜带来的结果是严重的，一种是宇宙的自然律所带来的；一种是神在祂约的关系中所做出的行动。前者就如诗篇的作者所说：造他的要和他一样；凡靠他的也要如此（诗115:8）。如果神是生命的活水源泉，那么离开了这活水的源头，我们的生命必然会干涸、枯竭。威斯敏神学院著名的新约学者G.K.Beale写了一本书，书名就叫：我们会成为他们所敬拜的。中国的古话：近朱者赤，近墨者黑，可以说明这一点。后者就如以色列的先知，控诉以色列百姓说，他们离弃了以色列的神，所以神就以战争、饥荒、瘟疫管教他们。
因此，神在圣经中，屡次严严的警戒祂的百姓，要远离一切的偶像崇拜（出20:4, 林前11:14）。

对古代的人来说，偶像崇拜是直接而明显的，祭祀就是直接的表达。对21世纪的现代人来说，祭祀某个神祗却明显是很愚昧、愚蠢的事情。现代的人怀疑灵的世界，不认为这些宗教祭祀会真的带来什么影响。但这并不意味着，现代人没有偶像。而是，这些偶像以隐晦的方式藏在现代人的心灵中。

海德堡要理问答，改革宗信仰最重要的告白之一，在解释十诫的第一诫的时候说到：偶像崇拜就是除了信靠在圣经中自我启示的唯一真神以外，另外有所信靠的。这不是指离弃神，而是说你在信靠神的同时，还信靠其它与神同等的东西。这些东西对现代的基督徒来说，可能是金钱、爱情、名声、成功或者服事本身。

####接受生肖是偶像崇拜吗？
我们知道生肖的起源和来历，也明白圣经所说的偶像崇拜是什么。回到我们一开始的问题，用生肖去称呼某一年，或者接受生肖属相能被定罪成偶像崇拜吗？我的答案是视情况而定。

  1. 如果你只是把生肖当做一个标记，不赋予它什么特殊的内涵，包括与之相关联的星座、运势、风水、宜忌等，个人以为就没有任何问题。即使生肖的起源可能是偶像崇拜，但在长时间的演变中，已经变成一种文化习俗，已经没有偶像崇拜的内涵，并不能被当做偶像崇拜。
    不然，如果这样的话，中国基督徒就什么节都不能过了，什么习俗都不能参与了。拿春节来说，过年跟偶像崇拜有关，贴春联跟偶像崇拜有关、放鞭炮跟偶像崇拜有关；甚至吃饺子也是。
    还有，圣诞节也不能过了，因为这一天原来是一个异教的节日，很可能是初代教会为了转化当时的异教风俗而做的一个决定。
    更糟糕的是，按着这样的逻辑，我们得强烈建议英文世界的基督徒不要用Monday周一、Tuesday周二…Sunday周日这些名字，因为这些名字都是以古代异教之神的名字来命名的。Monday是月亮神，Tuesday是战神…Sunday是太阳神。

这种以起源不正来进行反对论证的逻辑是行不通的，这使得基督徒基本上要完全摈弃原来的文化传统，重新创造一套自己的。但我们被呼召更多的是去更新我们所在的文化传统，而不是去创造一套全新的（当然有时候这也是需要的）。

  1. 但如果你接受了生肖背后的观念，以为生肖年会影响你在某一年的工作、生活，而属相会影响你的性情、运势，那么这些是偶像崇拜，是需要被拒绝的。因为知道我们每一天的日子乃是在那位创造我们，并在基督里爱我们的神的手中，而不是由你的出生时间所决定的。离开了神，所有的人都活在罪的阴影和咒诅之下，当然就没有什么所谓的良辰吉日；而在神的恩典中，即使在人看来遭难、黑暗的日子也可以是充满欢喜和祝福的。
    另外，从圣经的启示来看，时间是上帝创造的，有个起点也有个终点。时间更像一条射线，朝着神所定的目标前进；而不像一个圆圈，周而复始的循环。因此，中国传统的干支纪年背后的时间观——看历史为60年为一个循环，并不符合圣经的观念，应该加以摈弃。

#####基督徒怎么办？
所以，面对第2种情况，这属于偶像崇拜，如保罗所教导哥林多教会的弟兄姊妹，我们应该“远离一切的偶像崇拜”（林前10：14）。

而面对第1种情况，则接受生肖的说法本身不应该被定罪为偶像崇拜，但我们仍要记住保罗对哥林多教会的勉励：”凡事我都可行，但不都有益处；凡事都可行，但不都造就人（林前10:23）”。这个时候我们更需要是智慧，是爱心。我们要考虑的不是这件事对不对，而是这件事会不会叫其他的基督徒跌倒？我比较开放自由，但我是否愿意为了他人良心的缘故而限制我的自由，在这里特别是嘴唇。如果我观点保守，我是否能在基督里接纳别人比较包容的观念？

但不管是保守还是自由，我们不是属于任何的受造之物，包括十二生肖；我们是单单属于耶稣的。因为耶稣已经在十字架上为我们舍了生命，并且第三天复活。我们的旧人已经同祂钉十字架，现在活着的不再是我，乃是基督在我里面活着！

不管是保守还是自由，按着常人的观念，我们属什么，无论是鼠、牛、虎、兔…并不重要，重要的是我们是属于耶稣的。这是神在基督耶稣里所赐我们宝贵的身份。这是我们对自己、也对我们的孩子要教导的。让我以我女儿2岁左右经历的小故事来结束。

我和妻子不喜欢生肖属相的说法（再次说，这是个人喜好），也不希望我们的女儿被人说成是属虎的（她是2010年出生）。小家伙第一次突然被人问到”你属什么的？”这个问题时，因为不明白也不知如何回答，就跑到妈妈身边，眼泪快掉下来了。后来妈妈了解事情经过后，就告诉她，”如果有人问你属什么的，你就说属耶稣的。”小家伙知道了答案，又开心又踏实。在那之后，每当在电梯里或小区里有人问她这个问题时，她都会毫不犹豫，自信又自豪地说：”我是属耶稣的！”问的人这时都会有点困惑地把脸转向家长。家长就可以顺势和他谈谈信仰了。

谁知道呢，说不定，生肖的问题还可以是个绝好的福音切入口，让我们可以向那些不信的家人、朋友见证说：我是属耶稣的，祂是爱我，为我舍己！

未命名

发表于 2017-01-25 | | 阅读次数

阿爸父神，当我们踏进新的一年，向你承认我们需要你的同在以及面对未来的指引。

在这新的一年，我们每一个人都有自己的希望和期待——但只有你自己知道前面等待我们的是什么，也只有你能给我们够用的力量和智慧去面对所有的挑战。

因此，我们恳求你，帮助我们谦卑的跟随你的引领，并寻求你在这一年对我们生命的旨意。

面对未来所有的不确定，求你使我们确信你那永不改变的爱。
面对生活中无法避免的失落和伤痛，求你使我们转向你，好得着所需要的稳妥和安慰。
面对生命中的试探和顽梗自我意志的拉扯，求你帮助我们不至迷失方向，反而能勇敢的去做你认为对的事，无论那代价有多大。
我们的心思每天被各种事务和内心的渴求占据，求你让我们看到这个破碎世界的伤痛和不义，使我们以热情和牺牲的爱去回应那些孤单没有朋友的人和那些贫穷有需要的人。让我们可以像诗人那样祷告：“耶和华啊，求你将你的律例指教我，我必遵守到底！”（诗篇119:33）

我们为我们的国家和在这个充满挑战的时代治理国家的政府官员祷告，也为那些在这个充满危险和艰难的环境中努力寻求和平和正义的人祷告。特别与他们的家人同在，让他们能体会到你的爱和眷顾。
求你使我们这个支离破碎的社会能够合一，并给我们更大的异象看到你在我们身上所要成就的。你的话语提醒我们说：以耶和华为神的那国是有福的。

当我们回顾过去的这一年，我们感谢你一切的良善，远超过我们所该得的。让我们永远都不要把你过去的良善当做理所当然，或者忘记你所给我们的怜悯；而是因着你的良善和怜悯，我们更深的悔改，并更深的委身给你；在这新的一年，愿你成为我们生命的根基和中心。

父啊，为着你在这新的一年给我们的应许和盼望感谢你，因此我们带着期盼和信心进入这一年。所有这一切都是奉救主耶稣的名祷告，正是因着祂的受死和复活，我们得着今生和来世的盼望。阿门！
