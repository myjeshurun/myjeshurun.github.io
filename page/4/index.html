未命名

发表于 2017-01-21 | | 阅读次数

在巴底买的祈求中：大卫的子孙耶稣啊，可怜我吧，巴底买是一个被动的角色，耶稣是主动的，对吗？而在耶稣的回应里，耶稣把巴底买放在一个主动的位置上，而像一个仆人一样，对主人说：你要我为你做什么。耶稣降卑了祂自己，高抬了巴底买。耶稣在祂的降卑里面，进入了仆人的角色。在巴底买的认识中，耶稣是弥赛亚，是君王，是敬拜尊荣、恳求的对象；而在这里，耶稣是一个仆人，这个身份更能使他进入巴底买的生命中，与他的生命有相交。耶稣非常尊重我们，他常常在我们的内心深处问我们，你要我帮助你吗？只有当我们有回应之后，祂才会进到我们的生命中来。

耶稣的问题总是藏着玄机的，值得我们好好的思想。

  • 你们为什么找我呢？
  • 说“你的罪赦了”，或说“起来行走”，哪一样容易呢？
  • 人说我是谁？你们说我是谁？
  • 谁着落在那个强盗的人的邻舍呢？
  • 你们要什么？
  • 你要痊愈吗？
  • 雅各、约翰：要我为你做什么？

巴底买的故事常被当做是一个信心得医治的神迹来解读。耶稣的问题：要我为你做什么，也常常被简单的运用，好像上帝随便给我们开了一张空白支票似的。
背景是：耶稣三次预言自己要上耶路撒冷去受辱、被钉十字架，然后第三天复活。这是耶稣的使命，而且祂是很热切的要去实现上帝托付给祂的使命的。但这时候，祂站住了。

在信心中被医治
当一个人怀着很深的服事上帝的心，而同时又被自己的难处所困扰的时候，这个时候信心的祷告是很有力量的。我听到好多这样的见证，当人们祷告说：神啊，你医治我，我就把自己奉献做传道人；神啊，你医治我的孩子，我就把他奉献给你做传道人。神真的就垂听，赐下了医治。

没有跟随服事耶稣的动机，光有信心而做出来的所谓的“病得医治”的祷告是没有根基的。而有了服事的动机，又带着信心，所做出来的祷告，不一定是“病得医治”的祷告。

  • 芬尼·克罗斯比的见证。

    她在八十岁所著的《我的自传》中说：
    “我自幼至今，深信慈爱的主藉着祂无限的怜悯和奇妙的旨意，使用这种方法将我分别为圣，使我直到今天还能有机会，作祂所指派给我作的事工。
    如果明天我可以看到世上的景色，我也不会接受这项治疗，你曾经听过一个失明的人像我这样谈吐吗？尽管我的失明是由于治疗上的失误造成的。
    虽然这是医生粗心大意造成的错误，但是神从不误事。我深信这是祂的美意要我在生理上过着黑暗的日子，以便我能更专一地、单纯地歌唱赞美祂，并激励别人也随着赞美祂。若我不是失明的话，我就会被其它美好的事物所吸引，我就不可能写出论千计的圣诗，让世界各地的人参与歌诵。”

  • 妻子占从的见证
    巴底买成了门徒的标本、榜样。

  • make a way to Jesus: 瘫子的四个朋友；血漏的妇人
  • 舍弃一切
  • 在路上跟随。
    巴底买的人生完全的被翻转！马可告诉我们，他一开始是在路上讨饭的，最后呢，不是继续回去讨饭，而是在路上跟随耶稣。这是一条怎样的路啊？去耶路撒冷的路，十字架的路…

所以，我们看到：他从坐着乞讨的生活，变成行走交托的生活；他离开之前的卑贱之地，踏上十字架的荣耀旅程；他从渴求依靠人的施予，变成仰望信靠天父的供给；他从乞丐的行列迈入圣徒的团契，从一个卑贱的人生蜕变成一个尊贵的人生。
这正是马可对他所牧养的教会的期待。他特别记下了瞎子的名字：巴底买。这是一个奇怪的名字：Bar是亚兰文“儿子”的意思，底买，是希腊文“荣耀、尊贵”的意思。而“底买”相对应的亚兰文的意思是“卑贱、屈辱”的意思。马可借着巴底买的信心见证呼召、勉励他的会众：真正的从灵里看见耶稣，认识耶稣是那位被钉十字架的弥赛亚，并且在这条十字架的路上跟随耶稣，从而从卑贱之子变为荣耀之子。这也是2017年我对大家的勉励。

弟兄姊妹，在快进入中国新年的时候，你向上帝求的是什么？你确信上帝会垂听你的祷告吗？我祈求我们每一个人都有这样的确信，如果不确信，你为什么还祷告呢？
祷告，更重要的是预备我们的心；是进入与神亲密的相交里。
想象当你来到主面前，耶稣问你说：你要我为你做什么？你会怎么回答耶稣？更重要的，你回答的时候，是带着什么样的动机？是带着跟随耶稣荣耀神的动机，还是为了自己的享受和荣耀？愿我们都能给出合神心意的答案！

总统是靠不住的！

发表于 2016-11-16 | 分类于 微信文章 | | 阅读次数

著名的旅美作家林达以此为名写了一本书，让人了解美国的政治是怎么一回事。在这本书里，他说到：“把“克林顿政府”读成“克林 顿行政分支”，而且知道什么时候要把“美国政府”也读成“美国政府行政分支”，这是理解美国政治制度和权力结构的入门之课。”（总统是靠不住的，01章）
了解内幕的人，知道美国的政治不是靠一个人能左右的，进入了这个圈子，就自然的会被里面的体系和规则所左右，轻易想要呼风唤雨，左右时局并不容易。或许这正是美国民主政治的益处，在权力制衡中最大限度的防止人心的罪恶。从这个角度看，所有的人，对总统的选举结果都不应该又太高的期望，也不需要过于失望。在一个民主国家里，人民是国家的主人，

不要对总统寄予厚望

其实，不用了解内幕，一个重生得救、明白圣经的基督徒很清楚：所有人都是犯了罪，亏缺了上帝的荣耀。所有人都是罪人，一个义人也没有。人的盼望，民族的盼望，国家的盼望不是在哪个有限又有罪的人身上，而是在永活的上帝身上。
