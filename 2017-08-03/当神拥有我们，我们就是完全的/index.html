当神拥有我们，我们就是完全的

发表于 2017-08-03 | 分类于 灵修笔记 | | 阅读次数

梅顿说：当神拥有我们，我们就是完全的……一切就已蒙福成为“属灵的”，甚至包括那些属物质的事。
这几天，我又陷入一种危机之中，我以为是危机，但真的是危机吗？我害怕自己自欺，明明不对，还要为自己找到一个借口；另外，我又不想变成一个无趣的人，一个没有激情梦想，失去追求的人。
这个危机是我又喜欢折腾工具和博客了，我说自己是一个工具控，看上一个大家都说好的工具，就很想把它摸透。从Sleeping Bear Dunes回来，先是心血来潮折腾git，github和vim；然后是leanote，还梦想着把这个开源项目做成基督徒的原创写作平台。我觉得 #工具 和 #内容 是互相促进的，就如读者和作者是互相促进的一样，很少有作者完全不顾读者的感受而写作的。感叹在美国有那么多好的工具可以使用，比如我现在用的这个DayOne写日记应用，就再次激起了我写日记的热诚。

昨天决定用hexo和github pages搭建一个静态博客，我最后放弃了leanote，因为它的博客还是不够美观，不够geek。因为下载、安装的过程出现了好多问题，我就一直折腾到快5点才去睡觉。我心里有些不安，但还好，我不确定是自己的灵性麻木了？

在这过程中，我发现一些问题是，

  1. 自己很想做出一些能够呈现出来的东西，比如一个博客网站；
  2. 自己缺少等候的心，总想要马上实现，不想等到第二天。但是人生中有些事情就是需要等待的，比如你所喜欢的那个人。
  3. 自己喜欢解决问题带来的满足感：I fixed it. I can fix it.

这几天让我再次燃起对编程的热爱，如果不是耶稣，我估计我会一头扎进代码的世界里。在网络开始火起来的时候，大概2002年我就曾经说，娶来net，嫁给java。

有些怪怪的，如果我的弟兄姊妹知道了，他们的牧师那么喜欢编程，他们会怎么想？会说我不务正业吗？牧师不能喜欢编程吗？不能着迷于属灵事物之外的其它东西吗？

但牧师不是要完全分别为圣归给神，以祈祷传道为念吗？他不应该弃绝世上的一切享受、喜好吗？他的心思意念，他的时间，他的热诚，不应该都在与神、与信仰相关的事情上吗？是吗？

但我控制不住的要自己玩一玩代码，折腾出一个属于我自己的好看的博客网站，这怎么办？我应该放下这个吗？

我没有答案…
梅顿的话给了一些安慰：当神拥有我们，我们就是完全的。我的完全不是我自己的道德水平，不是我的义行，而是因着基督被神拥有。神啊，感谢你，你拥有我这个破烂不堪的罪人。

我今天早上犹豫了一下要不要告诉我的妻子，跟她分享我的软弱，但我担心她会批评我不抓紧时间，教导我，或者不理解。最终还是没有说，有些遗憾。

ps：越来越喜欢用DayOne写日记了。这种用markdown写作，并自动把#变成标签的风格，特别棒。我就不需要去使用找鼠标了。这种把markdown语言融入软件设计的做法，使得一切都变得那么简洁，而我需要的就是用markdown写东西。
