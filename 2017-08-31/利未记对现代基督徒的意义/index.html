利未记对现代基督徒的意义

发表于 2017-08-31 | 分类于 圣经解读 | | 阅读次数



许多弟兄姊妹读利未记的时候，觉得读不下去：一方面是因为其中许多的献祭礼仪、程序，洁净的规条令人觉得遥远陌生又枯燥乏味，远不如五经中记载的许多故事精彩；另一方面是认为这些礼仪、规条是旧约的律法，是过时的，对今天活在新约之下的基督徒已经不适用了。当我们这样来看利未记的时候，就是误解了利未记，而且错失了利未记作为神的启示的鲜活意义。

利未记是摩西五经的第三卷，是最中间的那一卷书；从位置来看，这卷书是整个摩西五经的核心。如果把摩西五经当做是旧约信仰的根基的话，那么利未记就是这个根基的核心；利未记的重要性可见一斑。那么，我们如何来解读利未记呢？这些两三千年前的礼仪、规条于今天的你我有什么意义呢？

首先，要把利未记放在摩西五经的框架中来理解。简单的来说，创世记讲神对以色列先祖亚伯拉罕、以撒、雅各的拣选和应许：土地、子孙和蒙福的国度。出埃及讲以色列国或者民族的诞生，以神带领以色列百姓出埃及和神在西乃山赐下十诫为标志性事件；但出埃及记不是以以色列的自由解放为结束，而是以会幕的建造完成为结束，这说明出埃及只是一个开始；会幕是神居住的所在，借着所搭建的会幕，神住在祂的百姓中间了！神与以色列的故事才刚刚开始。
利未记是这个故事的延续，也是整个故事的核心和关键。借着会幕，神已经住在蒙祂救赎的百姓中间；神不是在天上，而是在地上的会幕里！那么，利未记要回答的关键问题就是：神的百姓该如何与这位神相处呢？神的百姓如何活在神的同在中呢？利未记所有的礼仪、规条都是在用实际的例子或者实物教材来教导以色列人如何来亲近这位活在他们中间的神。
民数记讲以色列背叛神，在旷野中漂流的经历。申命记讲神借着摩西重申祂的律法，主题是预备新一代的以色列人如何在应许之地生活，问题依然是：如何活在神的应许、同在之中。

比如，对以色列人来说，会幕不只是一个地方而已，会幕是神的居所。来到会幕，就意味着来到神的居所，来亲近神。对以色列人来说，会幕最大的功能就是献祭，以色列人是借着献祭来亲近神的。献祭是亲近神最主要的手段和方式，这对我们今天的基督徒来说，是一个常备忽略的真理。利未记提到5种祭：燔祭、素祭、平安祭、赎罪祭、赎愆祭；这些祭在基督耶稣已经成全了。在新约里，燔祭对应罗12:1，把自己完全奉献给神，保罗说这是我们理所当然的敬拜/服事；素祭，对应来13:15，感恩颂赞的祭，这是承认祂名之人嘴唇的果子；平安祭，就是交通祭、分享祭，对应约壹1:7，与弟兄姊妹团契相交，这是行在神的光中的表现；赎罪祭、赎愆祭，认罪、悔改，对应罗3:25,来10:12，我们乃是靠着耶稣的血才能进到神的面前。当我们亲近神，特别是主日敬拜神的时候，我们是否预备我们的心献上这些奉献、感恩、颂赞、团契以及悔改的祭呢？

另外，利未记的核心主题是圣洁，而圣洁是神最重要的特质。R.C 史保罗，一位改革宗神学家，提到说：没有别的神的品性被重复三次的，除了圣洁之外（赛6:3）。即使是所有人都喜欢而且看重的神的慈爱的属性，圣经也没有一个地方宣告神说：“爱,爱,爱”的。表面上，整本利未记都在以实际的礼仪、规条教导以色列人该如何生活在神的会幕中（对祭司而言），和会幕的周围（对以色列人来说），免得他们得罪神。但要知道，这一切背后的神学原则是：让他们认识到神是一位圣洁的神。弟兄姊妹，你是否也清楚的认识到神是一位”圣哉、圣哉、圣哉”的神呢？你是否在实际的生活中，过一个圣洁的生活，免得我们得罪这位圣洁的神呢？

最后，利未记把我们引向耶稣基督，这位神在末世赐下的终极会幕、终极祭物。会幕不再只是一个物质的存在，会幕变成了神自己，约翰福音宣告说：道成了肉身，住（原文：支搭会幕）在我们中间，充充满满的有恩典、有真理（约1:14）。利未记中最关键也是终极的问题是：如何活在这位圣洁的神的同在中？神的百姓以色列的历史告诉我们，没有人能够靠着律法活出圣洁，讨神喜悦。新约给了我们答案：唯有靠着耶稣基督一次献上永远有效的赎罪祭（来10:12），我们才能活在神的同在中，而且是至圣的同在中，就是至圣所中（来10:19）。这是否再一次激起我们对耶稣基督、对天父上帝的感恩和颂赞呢？！
