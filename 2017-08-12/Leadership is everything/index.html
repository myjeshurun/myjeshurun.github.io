【GLS17】Bill Hybels——领导力就是你的一切

发表于 2017-08-12 | 分类于 领导力资源 | | 阅读次数

以下是比尔海波斯在8月10日的信息。Below is the message from Bill’s message on Aug. 10th, 2017

如何塑造一个彼此尊重的团队文化？

在这个宗教、宗族越来越多元的社会中，人们越来越不容易彼此尊重。根据调查，1998年美国有25%的人在职场上经历老板的粗鲁对待；在2005年，这个数字达到了50%。现在更是严重。人们没有受到尊重，被粗鲁对待，效率下降50%。

作为领袖该如何应对呢？这是Bill牧师提出的10条法则：

  1. 建立榜样：区分彼此的不同，但不妖魔化彼此。
  2. 进行有效的对话，而不动怒。
  3. 不打断，不一言堂。
  4. 控制语调，不贬低对方。
  5. 保持礼貌。
  6. 不一成不变的看待别人；不用简单化的模型看别人.
  1. 马上道歉，而不是为自己辩解。
  2. 谨慎的表达自己的观点，同时又保持敞开的心。
  3. 说到做到。
  4. 为别人树立尊重别人的榜样。

  • 容忍是给胆小的人的，勇敢的人努力寻求去理解别人，而不是单单的忍受。

  • 现任教皇Francis的榜样：给与自己不同的人洗脚；推动不同信仰间的对话；饶恕别人。

  • 关于卸任：建立一个有关卸任的计划是非常重要的。

领袖的内在生命

领袖们是那些带领别人面对困难和挑战的人。真正的领袖拥抱挑战，欢迎挑战，而且在挑战中兴盛。领袖不停的寻找更大的异象，领袖的血因着各种挑战、事务而沸腾。但如果领袖的生命中，只有挑战，没有亲密的关系，那么这条领袖之路很快会走到终点。

Bill分享说，在他事工快速增长的时候，一个长辈找到他，约他去散步，与他进行了一番很长的对话。那个长辈很直接的告诉他：如果你再这样下去，你可能会继续获得成功。但有一天，当你回到家的时候，你的妻子、孩子都不在家等你了。那时候，你会厌恶你的成功直到死去。

Bill说，仅仅是财政上的成功应该要让领袖觉得无趣。上帝创造了你，给你领袖的才干，是要让你带来改变的人。你的人生应该有比获得现世的成功更大的梦想、异象。

而领袖的内在生命（包括与神、与家人朋友的亲密关系）是领袖的生涯能否持续下去的关键。但是许多领袖却因为忙碌的事工忽略了自己和内在生命，这是很危险的。每天花15分钟安静下来，来回顾和反省你的人生，你的领导力，是非常重要的。在这15分钟里面，思想我们到底是为什么、为谁在活。

在一天开始的时候，不要马上跳入工作的海洋；而要首先向神祷告，再一次把自己的人生降服在那位爱你的上帝手中。

不要挤压你的默想时间，这会使一个领袖的生命枯干，领导力枯竭。

同时，要花时间和家人、朋友在一起。他们是你生命中最亲近的人，是你活在这个世界上最重要的意义之一。家人和朋友反过来会极大的丰富领袖的生命。

领袖们，在你忙得不知道北的时候，有没有人约你去散步，给你提个醒？我希望你的生命中，有这样的长辈或者朋友！
