大卫真的厚待米非波设了吗？

发表于 2017-08-18 | 分类于 灵修笔记 | | 阅读次数

撒下9章讲述大卫扫罗的孙子、约拿单的儿子米非波设，他是双腿残废的人。圣经说大卫因好朋友约拿单的缘故厚待米非波设，不仅把他爷爷扫罗家的一切都赐给他，还让米非波设常常和他一起坐席吃饭。这个故事常常被人引用来证明大卫王的良善和慈爱，但记得东杰牧师曾经提到过，大卫这样的做法其实不像他的朋友约拿单那么单纯。常常让瘸腿的米非波设在耶路撒冷常常跟王一块吃饭，是高抬他，还是羞辱他呢？

我仔细阅读的时候，感觉东杰牧师的分析的确很有道理。这其中有些疑点：

  1. 整个故事的背景是大卫日渐强盛，统统征服了周围的仇敌之后。9:1点出了一个鲜明的对比：大卫的鼎盛和扫罗家的破败（剩下的人，言外之意是大部分都死光了）。这个时候的大卫（才）想起了，要因约拿单的缘故向扫罗家施恩。约拿单是大卫在患难之中的朋友和救助，米非波设却是大卫在鼎盛时期施予的小恩惠。当然，人们也可以说，其实大卫可以赶尽杀绝，但他没有；而且迟到总比不到好。

  2. 大卫王把米非波设从玛吉家带到耶路撒冷，看起来是恩待他、高抬他。但从米非波设的角度来看，未必是这样。他在玛吉家或许比在耶路撒冷更加的舒服和自由。

    1. 想想看，常常见到大卫的辉煌，会让他想起自己家族的没落，让他伤心自卑。
    2. 在耶路撒冷要应对各种纷繁复杂的政治斗争，这对米非波设来说是没有办法应付的，圣经两次强调米非波设是瘸腿的；说明他自己并无能力去承受这些。后来大卫被他的儿子押沙龙追杀离开耶路撒冷，米非波设因故没有跟上，回来就被大卫责备了。大卫并没有怜悯他的腿瘸，大卫要求的是对他的忠诚。
    3. 大卫并没有问米非波设自己愿不愿意，大卫只是以一个恩慈的价码强制的命令米非波设搬到耶路撒冷住。

  3. 大卫是让洗巴来照顾米非波设，而不是之前照顾他的玛吉一家。圣经记载说，洗巴很强悍，有15个儿子，20个仆人。从后来发生的故事来看，洗巴有可能真的毁谤了米非波设。如果这样的话，他对米非波设是不安好心的。洗巴可能只是因为大卫给予的好处（或者想得到大卫的好处）来服事米非波设，而不是真正的爱米非波设来服事他。
    如果大卫这么做是故意的，那么他不是真正的爱米非波设，要厚待他；反而可能是要监控他，向他炫耀。当然，有人可以说，大卫并不是这样的人，他的心意本来是好的，之所以让洗巴照顾米非波设是因为洗巴有能力，能照顾的更好。
    但即使他的心意是好的，要厚待米非波设，但如第2点所记，他所做的却不是真正的祝福了米非波设。

  4. 从后来发生的事情来看（撒下19章），大卫一听见洗巴说米非波设没有来迎接他，是因为他想恢复他爷爷的荣耀；立马就废了他，把他的财产归给了洗巴。可见，大卫并没有真正的让米非波设拥有他所拥有的一切，大卫可以随时拿走。而他听到米非波设的忠心表白之后，又改口说，让他和洗巴平分产业；而没有追究洗巴毁谤主人的罪。可见，大卫并没有真正的站在米非波设的立场上，并不完全信任他。如果从这里往前推，大卫对米非波设存有戒心的话，那么大卫表面上是在厚待米非波设，其实是在软禁他。让瘸腿的米非波设常常和他一起吃饭，活在自己的眼皮底下，是最好的防止他叛乱、扫罗王权死灰复然的方式了。
    当然，大家也可以米非波设的回应来看大卫的做法：

    仆人算什么，不过如死狗一般，竟蒙王这样眷顾。（撒下9:8）

    只是在王宫的政治斗争中，很多话是不能按着表面来理解的。米非波设完全有可能是了解大卫意图的，他在那种情境中，除了正面的回应，没有别的出路。他后来也是这么捧大卫的：

    但我主我王像　神的使者一样，现在，你看怎样好，就怎样行吧！因为我祖父全家的人，在我主我王面前都算不得什么，都不过是该死的人；你却使仆人在你席上与你一同吃饭。我还有什么权利再向王哀求什么呢？”…..米非波设对王说：“我主我王既然已平平安安回宫，所有的一切就都任凭洗巴拿去好了。”（撒下27-28，30）

    如果米非波设像他父亲约拿单对大卫感情忠诚、单纯的话，也真真映衬出大卫的城府和心机了。

  5. 许多人觉得大卫的堕落从撒下11章开始，但在我看来在神使大卫大大得胜之后的第9章，大卫的身上马上就显出了罪的影响力。他并没有真正的用恩慈和良善对待米非波设，反而是用权力和强盛压制他；从这里显示了他对上帝的不信任。

  6. 指出这些，是要让大家看到大卫是真实的一个人，一个白手起家，靠着自己的能力登上王位宝座的人；他的城府和心机在整个的政治斗争中是难免的。《明朝那些事》的作者，就直接了当的说：对皇帝这个职位来说，绝大多数的情况下，好人做不了好皇帝。
    因此，即使大卫也有很多的缺点，这并不妨碍圣经评价他说，他是合神心意的人。他也是我们要效法的榜样，不是在他做人的事情上，而是他与神交往的事情上：他把神常摆在自己面前，他依靠神，他知错就改。
