高中毕业20周年聚会分享

发表于 2019-08-03 | 分类于 无题·随笔 | | 阅读次数

谢谢给我一个机会欢迎所有参加这次20周年聚会的人，特别是那些还没有打过招呼的。
我相信大家跟我一样特别期待今天的聚会，有没有昨天晚上睡不着觉的，哈哈。我知道有别的原因睡不好觉的。然后，我们真的在这里，见到彼此的时候，心里真是好激动。我们今天在这里是共同创造高三4班的集体记忆。

真的要谢谢这次筹委会的每个人，特别谢谢驻守家乡的剑伟、东辉、草鱼、伟达等，你们使许多人心中的期待成为现实。也谢谢来参加的老师们，谢谢你们陪伴我们走过了人生的三年青春；谢谢每一个紧张忙碌中抽出时间参加聚会的同学，我们今天在这里是共同创造高三4班的集体记忆。我忍不住要说：感谢上帝。但让我们为所有人再次鼓掌吧！

毕业20年了！
有些人，真的有20年不见了；见到许久不见的同学，相信大家都有一种恍若隔世的穿越感。在这个时空穿越中，谁敢说自己一点儿好奇心都没有：那个人到底怎么样了？哪个人？就是那个，你上铺的那个兄弟，一起打球的那个哥们，那个不可一世的，那个老实巴交的，那个一起通宵玩牌的，那个曾经跟你干架的，还有那个曾经让你害羞脸红、心跳加速的，那个，那个人后来怎么样了？如果你没有这份好奇，我会怀疑你是否来对了地方，哈哈。我想这是同学会最美妙的地方。我衷心盼望这份好奇心的满足对所有人都是一个祝福。

对大部分人来说，这会是人生的黄金20年，不仅是成家立业，生儿养女；还有这20年丰富多彩的生活阅历。这大概是我们人生最跌宕起伏、腥风雪雨的20年。在20年，我们尝到了许多的酸甜苦辣，学到了许多为人处事的道理；在这20年，我们更认识了这个社会，更认识了别人，也更认识了我们自己。不管是岁月改变了我们，还是我们改变了岁月。总之，今天的我们跟以前不一样了。我衷心盼望，我们每个人在这次的怀旧聚会中再次看到自我生命的进化更新。

如果把我们人生比作一场球赛，篮球赛吧，过去的20年加上之前的20年，好比比赛的前奏和上半场；现在大概到了中场休息的时候。无论上半场的战绩如何，我们知道比赛尚未结束，还有下半场、还有谢幕。只是下半场跟上半场不太一样，各方面都不太一样。如果上半场重在成就，那么下半场重在意义。上半场更多地问做什么和如何，下半场却更多地问为什么。

最近读李开复写的《AI·未来》，作为人工智能领域顶级研究员，他在53岁的时候，在他的事业不断迈向巅峰的时刻，查出了淋巴癌晚期，活过5年的几率只有50%。在残酷的现实面前，他突然意识到：以前自己疯狂工作使机器能像人一样思考，结果自己却成了像机器一样思考的人。他发现自己生命中对妻子对孩子对孩子的亏欠。他突然意识到生命的本质不是工作，不是外在的成就，而是爱。

因此，我也多少有些期待，在这次20周年对过去的回忆和追寻中，在彼此的欢笑和交谈中，我们人生的价值和意义或慢慢浮现、或更加清晰。请允许我用这样的话来结束：愿上帝祝福我们！

我们活在一个加速发展的时代，我建议我们聚会的频率也要加速，20年太久，至少得10年一次。所以，我提议我们10年后再聚首。
