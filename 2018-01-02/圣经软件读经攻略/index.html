Youversion 新功能：与朋友共读圣经

发表于 2018-01-02 | 分类于 工具控 | | 阅读次数

在这2018 年刚开始的时候，许多弟兄姐妹很想更好的学习神的话，就是圣经。笔者特向你介绍一款精致华丽又实用高效的圣经软件Youversion Bible App (圣经软件)。该圣经软件是全世界最受欢迎的圣经软件，它免费提供1134 种语言的1588 个圣经译本，全球安装数量超过3 亿。它功能丰富，不仅是一款帮助你阅读圣经、查询经文的好帮手，也可以帮助你藉着各种灵修材料，默想神的话；同时，因着网络技术的发达，这款圣经软件可以形成一个庞大的虚拟社区，这个社区因着神的话而紧密连接在一起。而这一点可能成了这款圣经软件的独一无二的亮点。

其他的圣经软件可能有一些独特的功能，但在网络读经社区的建设上，没有其他圣经软件可以超越Youversion。这一点对那些喜欢和别人一起读经，和别人分享读经心得的人意义重大，本文特别就这个独门绝技做一个全面的介绍(主要以安卓手机为例)。

  1. 下载Youversion 圣经软件
    • 海外用户：在苹果应用商店App Store或谷歌应用商店Play Store，搜索：Bible 或Youversion 即可下载
  • 国内用户：(请非比说明一下这个～～)

  1. 注册用户、登陆

    • 打开的应用界面下方，有五个小按钮：，从左到右分别是：首页，圣经，读经计划，账户，设置；点击账户“我”这一选项，就会出现“登录””注册”账户的界面。

    • 如果你已经注册了账号，就直接“登录”；如果没有，点击“注册”，进入“注册”界面，这里有三个选项：谷歌、Facebook 和电子邮件。对国内的用户来说，因为谷歌的服务和Facebook 都被屏蔽了，除非翻墙，唯一的方式是使用邮箱注册。

    • 输入你的名、姓，个人邮箱和密码，点击“注册”即可。小菜^^

  2. 搜索朋友
    登录之后，可以看看你的朋友中都有谁也在使用这个圣经软件～

    • 点击下方账户小人“我”标签，进入账户界面，在界面上方，点击”朋友“

    • 在接下来打开的界面上方，点击”添加朋友“

    • 这里提供了三种方式，让你和朋友建立连接：1）手机联系人，2）Facebook 社交网络，以及3）搜索朋友姓名。大家可以自行试试前两种方式添加朋友，我们在这里介绍搜索的方式

    • 点击界面右上角的”搜索“按钮之后，会出现一个文字框，让你输入。在这里你可以输入你朋友在该应用上的注册邮箱和使用的姓名，可以是中文名字或者英文名字。这是用户在注册时提供的信息，见上面第2 步。

  3. 使用读经计划
    读经计划有诸般好处：

    1. 克服人的惰性和遗忘，透过做计划和每日的提醒帮助我们每天来聆听神的话语。这对忙碌的现代人来说，是很重要的。
    2. 不同的读经计划帮助我们从不同的角度、让我们带着不同的目的去阅读的话语。比如很有名的麦琴读经表，一方面让人可以每年读一遍旧约，两遍新约和诗篇；另一方面让人可以跟踪神在历史不同时期的救赎进程。这款圣经软件提供了许多主题的读经计划，比如：婚姻，恋爱，医治、赦免等，这些主题的读经计划其实是在帮助人系统的查考圣经在该主题的教导。
    3. 有些读经计划与每日灵修紧密结合在一起，这样就不仅是读了圣经，还可以深入默想经文在我们今日生活中的应用。读经和灵修默想是信徒借以亲近神的主要路径。
      ”每日箴言“是普世佳音机构出版的灵修材料，中文版已经有30年的历史。该材料每月围绕一个主题，帮助信徒深入的默想。如今它也提供了与灵修配套的读经计划。在这里就以此为例，向读者介绍如何在youversion 圣经软件上使用读经计划。

    • 点击主界面下方中间的“计划”标签，进入“计划”界面。你会看到最上方有两个Tab按钮：我的计划和发掘计划。

    • 点击“发掘计划”，你会看到上面所说的许多圣经计划；点击右下角浮动的“搜索”按钮；

    • 在出现的搜索框中，输入关键字：每日箴言，你就会看到每日箴言的读经计划。如果发现找不到中文计划，请退出该界面，点击下方的“设置”按钮，在“语言”下拉菜单当中选择“简体中文”。

    • 点击进入“每日箴言”，选择“开始计划”；这时候，你可以选择“独自阅读”或“和朋友共读”。在这里，我们选择“和朋友共读”进入下一步。

  4. 邀请朋友一起读经
    和朋友一起读经能帮助我们坚持读经。我们每个人都有意志软弱和懈怠的时候，和朋友在一起就能彼此鼓励。

  • 如果你遵循以上的步骤，你找到了“每日箴言”的读经计划，选择“和朋友共读”，你会进入该读经“计划信息”，你确定哪一天开始该计划，接着你选择和谁一起读经。

  • 如果你还没有想好和谁一起读经，那么先不选，可以以后再发送链接给别人，让别人加入。

  • 在打开的界面底部，点击文本框右侧的分享按钮，把链接分享给别人；别人点击链接就可以接受邀请。

  • 别人接受邀请之后，就可以开始读经小组了。当你的Youversion朋友接受邀请后就可以在每天阅读的页面看见你的朋友们是否阅读了:-)

6.发表读经心得
一起读经不仅为了是彼此督促，更重要的是可以彼此分享读经的感动和收获，以彼此激励。如果你选择了“和朋友共读”的方式，在该读经计划的界面中，除了显示该日读经的内容：灵修内容和经文；还有一个“主题讨论”，用于该读经小组成员分享感动和收获：从今天的读经中，试举一件上帝要告诉你的事。这个功能实在了不起，最大程度的模拟了读经/灵修小组的功能。
我还记得在大学团契的时候，每天早上弟兄姐妹相约聚在学校的某个角落一起灵修。大家安静之后，各自读经默想；然后大家轮流分享读经的感动：今天的经文中，哪些话最触动你？或者，上帝透过今天所读的经文对你的心说了什么？或者，你今天该如何回应上帝的话语？

不仅如此，该应用还提供了点赞功能好回应别人的分享，虽然很简单。如果你的朋友在读后记了笔记，别忘了点赞。我们自己也可以记读后感让别人点赞！在这种相互的分享、点赞当中完成相互的监督是这款圣经软件的精妙之处！

在已经来临的2018 年，愿我们每一个人都能在神的话语中成长。透过神的话语，不仅自己成长，更和别人一起成长！
